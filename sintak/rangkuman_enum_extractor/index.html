Implementasi EnumExtractor Trait

Panduan ekstraksi nilai string murni dari BackedEnum guna menghindari *Fatal Error* konversi objek di Ajocell v2.

Sensor Polymorphic Layer

Metode extractEnumValue dirancang kebal 100% dari kegagalan sistem data dengan melewati 5 lapis sensor validasi tipe data sebelum mengeluarkan output string murni:

1. Null Guard Jika data kosong/null, langsung return null tanpa diproses.
2. Collection Sensor Jika berupa kumpulan model, otomatis mengupas data item pertama.
3. Eloquent Raw Membaca original value database sebelum diubah oleh Mutator.
4. BackedEnum core Mengambil properti ->value asli dari enum PHP.
5. Safe String Cast Fallback terakhir melakukan paksa casting string secara aman.

1. Penerapan pada API Resource / Controller Transformer

Memastikan output JSON hanya berisi string murni untuk dikonsumsi JavaScript/React.

ProductResource.php 
 $this->id,
            'product_name'=> $this->name,
            
            // 🎯 CONTOH NYATA EKSTRAKSI FIELD:
            // Mengubah BackedEnum (misal: ProductType::PRABAYAR) menjadi string murni "prabayar"
            'product_type'=> $this->extractEnumValue($this->resource, 'type'),
            
            // Mengubah status transaksi enum menjadi string murni "success", "pending", atau "error"
            'status'      => $this->extractEnumValue($this->resource, 'status'),
        ];
    }
}
