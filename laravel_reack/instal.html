Inisialisasi Projek & Setup Database

Langkah-langkah awal membangun projek Buku Guru menggunakan Laravel 12, Breeze (Inertia-React), dan konfigurasi environment.

1 Create Project & Install Breeze 

# Membuat projek baru
composer create-project laravel/laravel buku-guru

# Masuk ke direktori
cd buku-guru

# Install Laravel Breeze untuk sistem Auth
composer require laravel/breeze --dev

2 Install Scaffold Inertia-React 

# Install Breeze dengan stack React
php artisan breeze:install react

# Install dependencies frontend
npm install

3 Install UI Packages 

# Install SweetAlert2 untuk notifikasi popup
npm install sweetalert2

# Install Lucide React untuk icon
npm install lucide-react

4 Setting Database (.env)

Buka file .env dan sesuaikan konfigurasinya: 

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=buku_guru
DB_USERNAME=root
DB_PASSWORD=

5 Migrasi & Running 

# Jalankan migrasi database
php artisan migrate

# Jalankan server frontend (Vite)
npm run dev

# Jalankan server backend (Laravel)
php artisan serve

Database Reset (Hapus All)

Gunakan ini jika ingin mengosongkan database dan menjalankan seeder ulang: 

php artisan migrate:fresh --seed
