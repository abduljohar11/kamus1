Setup Development Environment

Konfigurasi khusus Linux Mint & Manual Windows Setup

Linux Mint: Menggunakan nala untuk tampilan instalasi yang lebih rapi dan cepat.

1. Install PHP via Nala

sudo apt install nala -y
sudo nala update
sudo nala install php php-cli php-fpm php-mysql php-zip php-gd php-mbstring php-curl php-xml php-bcmath php-intl -y

2. Install Composer (Global)

curl -sS https://getcomposer.org/installer | php
sudo mv composer.phar /usr/local/bin/composer
composer --version

3. Install Node.js (LTS Version) 

curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo nala install nodejs -y
