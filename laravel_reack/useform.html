State & Form Management

Memahami penggunaan useForm sebagai standar industri untuk mengelola input, validasi, dan pengiriman data ke Laravel.

01 Apa itu useForm?

useForm adalah helper hook eksklusif dari @inertiajs/react. Ia menggabungkan fungsi useState dan router menjadi satu paket yang mengelola siklus hidup formulir secara otomatis.

02 Implementasi Kode (Logic) 

// 1. Import hook dari library
import { useForm } from '@inertiajs/react';

export default function CreatePost() {
    // 2. Destrukturisasi objek useForm
    const { data, setData, post, processing, errors, reset } = useForm({
        title: '',
        content: '',
    });

    const submit = (e) => {
        e.preventDefault();
        
        // 3. Kirim data ke endpoint Laravel
        post('/posts', {
            onSuccess: () => reset(), // Bersihkan form jika berhasil
        });
    };

    // ... return JSX ...
}

03 Atribut Utama useForm

Atribut Fungsi
data Objek penampung input user (Real-time).
setData Fungsi untuk mengupdate 'data' per field.
processing Status Boolean (Loading). Mencegah double-click.
errors Menampung pesan kegagalan validasi dari Laravel.
post/put/delete Metode pengiriman data (HTTP Methods).

🛡️ Catatan Cyber Security

Selalu gunakan atribut disabled={processing} pada tombol submit. Hal ini mencegah serangan sederhana berulang (Spam) dan menjaga integritas database agar tidak terjadi duplikasi data akibat ketidaksabaran pengguna.
