Menampilkan Detail Data Post

  • Langkah 1 - Menambahkan Method Show
  • Langkah 2 - Membuat Halaman View Show
  • Langkah 3 - Uji Coba Detail Data Post

Pada materi sebelumnya kita telah berhasil melakukan proses insert data ke dalam database menggunakan model dan Eloquent. Sekarang kita akan lanjutkan belajar menampilkan detail data post berdasarkan ID yang diambil dari URL browser.

Langkah 1 - Menambahkan Method Show

Sekarang kita akan menambahkan method baru yang bernama show di dalam controller PostController.

Silahkan teman-teman buka file app/Http/Controllers/PostController.php, kemudian ubah semua kode-nya menjadi seperti berikut ini.

<?php

namespace App\Http\Controllers;

use App\Models\Post;
use Illuminate\Http\Request;

class PostController extends Controller
{
    /**
     * index
     *
     * @return void
     */
    public function index()
    {
        //get posts
        $posts = Post::latest()->paginate(5);

        //render view with posts
        return view('posts.index', compact('posts'));
    }

        /**
     * create
     *
     * @return void
     */
    public function create()
    {
        return view('posts.create');
    }

    /**
     * store
     *
     * @param Request $request
     * @return void
     */
    public function store(Request $request)
    {
        //validate form
        $this->validate($request, [
            'image'     => 'required|image|mimes:jpeg,png,jpg,gif,svg|max:2048',
            'title'     => 'required|min:5',
            'content'   => 'required|min:10'
        ]);

        //upload image
        $image = $request->file('image');
        $image->storeAs('public/posts', $image->hashName());

        //create post
        Post::create([
            'image'     => $image->hashName(),
            'title'     => $request->title,
            'content'   => $request->content
        ]);

        //redirect to index
        return redirect()->route('posts.index')->with(['success' => 'Data Berhasil Disimpan!']);
    }
    
    /**
     * show
     *
     * @param  mixed $id
     * @return void
     */
    public function show($id)
    {
        //get post by ID
        $post = Post::find($id);

        //return view
        return view('posts.show', compact('post'));
    }
}

Dari perubahan kode di atas, kita menambahkan method baru yang bernama show dan di dalamnya kita berikan parameter $id yang nantinya diambil dari URL browser.

public function show($id)
{

	//...
	
}

Di dalam method tersebut kita melakukan get data post berdasarkan ID yang diambil dari parameter $id.

//get post by ID
$post = Post::find($id);

Setelah data berhasil diambil, kita kirimkan data tersebut ke dalam view show yang ada di dalam folder resources/views/posts.

//return view
return view('posts.show', compact('post'));

Langkah 2 - Membuat Halaman View Show

Setelah data dikirimkan oleh controller, maka kita tinggal menampilkannya pada halaman show. Silahkan teman-teman buat file baru dengan nama show.blade.php di dalam folder resources/views/posts, kemudian masukkan kode berikut ini di dalamnya.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Detail Data Post - SantriKoding.com</title>
    <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
</head>
<body style="background: lightgray">

    <div class="container mt-5 mb-5">
        <div class="row justify-content-center">
            <div class="col-md-8">
                <div class="card border-0 shadow rounded">
                    <div class="card-body">
                        <img src="{{ asset('storage/posts/'.$post->image) }}" class="w-100 rounded">
                        <hr>
                        <h4>{{ $post->title }}</h4>
                        <p class="tmt-3">
                            {!! $post->content !!}
                        </p>
                    </div>
                </div>
            </div>
        </div>
    </div>
    
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
<script src="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>
</body>
</html>

Di atas, untuk menampilkan detail data post kita bisa seperti berikut. Contohnya untuk data title.

{{ $post->title }}

Karena data content akan berisi sintaks-sintaks HTML, maka untuk menampilkannya kurang lebih seperti berikut ini.

{!! $post->content !!}
