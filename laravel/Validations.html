Validations

  • Membuat Validasi
  • Menampilkan Error Validasi
  • Aturan Validasi Yang Tersedia

Validation merupakan hal yang sangat penting saat membuat aplikasi. Validation merupakan metode yang digunakan untuk memvalidasi permintaan HTTP yang masuk dengan berbagai aturan validasi yang kuat.

Secara default class controller di dalam Laravel memiliki Traits yang bernama ValidatesRequests, sehingga kita bisa menggunakan validasi di dalam controller secara langsung.

Membuat Validasi

Berikut ini contoh penulisan validasi di Laravel.

public function store(Request $request)
{
    $request->validate([
        'title' => 'required|unique:posts|max:255',
        'body' => 'required',
    ]);
 
    // The blog post is valid...
}

Di atas, kita bisa melakukan define validasi menggunakan method validate dan di dalamnya kita berikan validasi yang diinginkan.

Menampilkan Error Validasi

Untuk menampilkan validasi, kita harus menulis kode-nya di dalam view, berikut ini contoh untuk menampilkan error validasi pada halaman view.

@error('title')
    <div class="alert alert-danger">{{ $message }}</div>
@enderror

Di atas kita menggunakan directive @error dan di dalamnya berisi key validasi yang di definisikan di dalam controller. Dan untuk menampilkan pesan errornya, kita menggunakan $message.

Aturan Validasi Yang Tersedia

Untuk list rule / aturan validasi yang tersedia di dalam Laravel, teman-teman bisa melihatnya melalui link berikut ini : https://laravel.com/docs/9.x/validation#available-validation-rules
