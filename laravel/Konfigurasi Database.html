Konfigurasi Database

  • Langkah 1 - Konfigurasi Koneksi Database
  • Langkah 2 - Membuat Database

Karena akan bekerja menggunakan database, maka kita perlu melakukan konfigurasi koneksi database-nya terlebih dahulu.

Langkah 1 - Konfigurasi Koneksi Database

Silahkan buka project kita menggunakan Text Editor, kemudian cari file yang bernama .env.

Jika sudah ketemu, silahkan cari kode berikut ini di dalam file .env.

DB_CONNECTION=sqlite
# DB_HOST=127.0.0.1
# DB_PORT=3306
# DB_DATABASE=laravel
# DB_USERNAME=root
# DB_PASSWORD=

Dan ubahlah menjadi seperti berikut ini :

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=db_laravel_untuk_pemula
DB_USERNAME=root
DB_PASSWORD=

Di atas, kita atur untuk DB_DATABASE atau nama database yang akan kita gunakan menjadi db_laravel_untuk_pemula.

Kemudian untuk DB_USERNAME kita biarkan default, yaitu root dan untuk DB_PASSWORD silahkan disesuaikan dengan konfigurasi-nya massing-masing. Jika menggunakan XAMPP, maka untuk password-nya dikosongkan saja (by default kosong).

Dan untuk DB_CONNECTION kita ubah menggunakan mysql.
