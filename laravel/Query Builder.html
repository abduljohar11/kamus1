Query Builder merupakan fitur di dalam Laravel yang bisa kita gunakan untuk menjalankan sebuah query ke dalam database dengan lebih mudah dan sederhana. Query Builder biasanya digunakan untuk menjalankan query yang sangat kompleks dan query yang susah ditangani oleh Eloquent.

Query Builder menggunakan sistem parameter binding yang ada di dalam PDO, sehingga akan lebih aman jika terkena serangan SQL Injection.

Menampilkan Semua Record

Berikut ini contoh menampilkan semua record yang ada di dalam table menggunakan Query Builder.

use Illuminate\Support\Facades\DB;


$users = DB::table('users')->get();

Di atas, kita harus import facades DB terlebih dahulu. Setelah itu, kita tinggal memanggil semua record menggunakan method get.

Menampilkan Sepsifik Field Table

Contoh sebelumnya akan memanggil semua field yang ada di dalam table users. Untuk memiliki field yang ditentukan, maka kita bisa menggunakan kode seperti berikut ini.

$users = DB::table('users')
            ->select('name', 'email as user_email')
            ->get();

Di atas, kita menambahkan method select dan berisi field yang ingin kita tampilkan. Di atas juga kita bisa melakukan inisialisasi nama field, contohnya email akan ditampilkan menjadi user_email.

Menampilkan Semua Record Dengan Pagination

Kita juga bisa menggunakan pagination di dalam Query Builder, contohnya seperti berikut ini.

use Illuminate\Support\Facades\DB;


$users = DB::table('users')->paginate(10);

Menampilkan Detail Data

Untuk menampilkan data, kita bisa menggunakan method first di dalam Query Builder. Contohnya seperti berikut ini.

$user = DB::table('users')->where('name', 'John')->first();

Join Table

Kita juga bisa melakukan join table menggunakan Query Builder dengan sangat mudah, contohnya seperti berikut ini.

use Illuminate\Support\Facades\DB;
 
$users = DB::table('orders')
            ->join('users', 'orders.user_id', '=', 'users.id')
            ->get();

Insert Record

Di dalam Query Builder untuk melakukan proses insert data ke dalam database kita bisa menggunakan method insert. Contohnya seperti berikut ini.

use Illuminate\Support\Facades\DB;

DB::table('users')->insert([
    'name'      => 'Fika Ridaul,
    'email'     => 'santrikoding@gmail.com',
]);

Kita juga bisa melakukan insert menggunakan format array, contohnya seperti berikut ini.

use Illuminate\Support\Facades\DB;

DB::table('users')->insert([
    ['name' => 'Fika Ridaul', 'email' => 'santrikoding@gmail.com'],
    ['name' => 'Akhmad Lutfi', 'email' => 'lutfi@gmail.com'],
    ['name' => 'Faisal Mahadi', 'email' => 'faisal@gmail.com'],
]);

Update Record

Selain memasukkan record ke dalam database, Query Builder juga dapat memperbarui record yang ada menggunakan metode update. Contohnya seperti berikut ini.

use Illuminate\Support\Facades\DB;

DB::table('users')
              ->where('id', 1)
              ->update(['name' => 'Fika Ridaul Maulayya']);

Update Or Insert

Method ini sangat membantu sekali, jadi dengan method ini kita bisa menjalankan 2 kondisi sekaligus. Jika record sudah ada di dalam database, maka kita akan lakukan update. Tapi jika belum ada maka akan lakukan proses insert. Contohnya seperti berikut ini.

use Illuminate\Support\Facades\DB;

DB::table('users')
    ->updateOrInsert(
        ['email' => 'john@santrikoding.com', 'name' => 'John'],
    );

Delete Record

Query Builder juga bisa melakukan proses delete data dari table menggunakan method delete. Contohnya seperti berikut ini.

use Illuminate\Support\Facades\DB;


DB::table('users')->where('id', 1)->delete();

Atau jika ingin menghapus semua record, kita tidak perlu menambahkan method where.

use Illuminate\Support\Facades\DB;


DB::table('users')->delete();
