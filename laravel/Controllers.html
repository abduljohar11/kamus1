Controllers

  • Controller Dasar
  • Single Action Controller

Controller merupakan salah satu pattern yang ada di dalam MVC, yaitu Model, View dan Controller. Controller umumnya digunakan untuk menangani sebuah kondisi tertentu. Misalnya di dalam controller kita gunakan untuk mengambil data, melakukan proses insert, update dan delete dari database.

Untuk membuat controller kita bisa menggunakan perintah make:controller dan semua file controller akan berada di dalam folder app/Http/Controllers.

Controller Dasar

Berikut ini kita akan coba membuat controller yang sederhana untuk menampilkan data user. Pertama kita akan membuat controllernya terlebih dahulu.

php artisan make:controller SantriController

Perintah di atas digunakan untuk membuat controller baru dengan nama SantriController.php yang berada di dalam folder app/Http/Controllers.

Kemudian kita akan coba berikan contoh sederhana untuk menampilkan text di dalam controller di atas.

<?php
 
namespace App\Http\Controllers;
 
class SantriController extends Controller
{
    public function index()
    {
    	return "SantriKoding!";
    }
}

Di atas, kita membuat method baru dengan nama index dan di dalamnya kita melakukan return ke dalam text SantriKoding!.

Controller di atas tidak akan berjalan, kecuali kita memanggilnya melalui route. Berikut ini contoh pemanggilan controller melalui route yang ada di dalam file routes/web.php.

<?php

use Illuminate\Support\Facades\Route;

//import controller "SantriController"
use App\Http\Controllers\SantriController;
 
 
Route::get('/santrikoding', [SantriController::class, 'index']);

Di atas, kita import class controller-nya.

use App\Http\Controllers\SantriController;

kemudian kita define route baru dengan path /santrikoding yang mengarah ke dalam SantriController dengan method index.

Kita bisa mengakses route di atas di dalam browser dengan http://localhost:8000/santrikoding.

Single Action Controller

Saat mengembangkan aplikasi mungkin kita akan menjumpai controller yang hanya memiliki 1 method saja. Alih-alih menggunakan controller biasa, kita bisa menggunakan yang namanya Single Action Controller.

Untuk membuat Single Action Controller kita bisa menggunakan perintah make:controller KodingController --invokable. Jadi seperti membuat controller pada umunya, hanya saja kita tambahkan flag --invokable atau -i.

Berikut ini contoh membuat controller dengan jenis Single Action Controller.

php artisan make:controller KodingController --invokable

php artisan make:controller KodingController -i

Kedua perintah di atas memiliki hasil yang sama. Dan hasil dari perintah di atas kurang lebih seperti berikut ini.

<?php
 
namespace App\Http\Controllers;
 
class KodingController extends Controller
{
    /**
     * Provision a new web server.
     *
     * @return \Illuminate\Http\Response
     */
    public function __invoke()
    {
        return "SantriKoding!";
    }
}

Saat membuat controller dengan jenis Single Action Controller, maka di dala class-nya akan terdapat method yang bernama __invoke. Method tersebut bisa kita gunakan untuk jenis kondisi apapun. Dalam contoh di atas kita gunakan untuk menampilkan text SantriKoding!.

Dan untuk memanggil controller di atas di dalam route, kita tidak perlu menuliskan spesifik method-nya. Kurang lebih seperti berikut ini.

<?php

use Illuminate\Support\Facades\Route;

//import controller "KodingController"
use App\Http\Controllers\KodingController;
 
 
Route::get('/koding', KodingController::class);

Kita bisa mengakses route di atas di dalam browser dengan http://localhost:8000/koding.
