Semua route dan controller harus mengembalikan sebuah response untuk ditampilkan kembali ke pengguna / user di dalam browser. Laravel memiliki beberapa cara untuk mengembalikan response. Response paling dasar adalah mengembalikan string dari route atau controller.

Contohnya seperti berikut ini.

Route::get('/', function () {
    return 'Hello World';
});

Di atas, kita mngembalikan response string Hello World.

Response Redirects

Response redirect merupakan turunan dari class Illuminate\Http\RedirectResponse. Dan digunakan untuk mengarahkan user ke URL lain.

Ada beberapa cara untuk menggunakan RedirectResponse. Paling umum adalah menggunakan helper redirect. Contohnya seperti berikut ini.

Route::get('/dashboard', function () {
    return redirect('home/dashboard');
});

Terkadang kita juga mungkin mengarahkan user ke halaman sebelumnya, biasanya digunakan jika sebuah form terdapat validasi yang belum terpenuhi, maka kita akan arahkan ke halaman sebelumnya. Disini kita bisa menggunakan helper yang bernama back. Contohnya seperti berikut.

Route::post('/user/profile', function () {
    
    // Validate the request...
 
    return back()->withInput();
});

Response JSON

Ketika melakukan response ke dalam format JSON, maka otomatis header Content-Type akan berisi application/json.

Contohnya seperti berikut ini.

return response()->json([
    'name' 		=> 'Fika Ridaul Maulayya',
    'company' => 'SantriKoding',
]);

Response Download

Response download dapat digunakan untuk menghasilkan response yang memaksa browser melakuakn download file dari URL yang diberikan.

Contohnya seperti berikut ini.

return response()->download($pathToFile);

return response()->download($pathToFile, $name, $headers);
