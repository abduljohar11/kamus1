Install dan menjalankan Laravel

  • Langkah 1 - Membuat Project Laravel
  • Langkah 2 - Menjalankan Project Laravel
  • Langkah 3 - Menjalankan Storage Link

Untuk membuat project Laravel baru, kita bisa menggunakan perintah composer create-project di dalam Composer. Dengan perintah tersebut kita bisa menginstall project Laravel dengan spesifik versi yang diinginkan.

Langkah 1 - Membuat Project Laravel

Langsung saja, silahkan teman-teman masuk ke dalam folder dimana akan menyimpan project Laravel-nya. Kemudian jalankan perintah berikut ini di dalam terminal/CMD :

composer create-project laravel/laravel:^11.0 laravel-untuk-pemula

Perintah di atas digunakan untuk membuat project Laravel baru dengan versi 11.x dan untuk namanya adalah laravel-untuk-pemula. Silahkan tunggu proses installasinya sampai selesai.

Langkah 2 - Menjalankan Project Laravel

Setelah berhasil melakukan proses installasi Laravel, sekarang kita lanjutkan belajar bagaimana cara menjalankan project tersebut. Silahkan jalankan perintah berikut ini di dalam terminal/CMD :

cd laravel-untuk-pemula

Perintah cd di atas, digunakan untuk melakukan navigasi atau masuk ke dalam folder project laravel-untuk-pemula. Jika sudah masuk di dalam folder project, sekarang jalankan perintah berikut ini :

php artisan serve

Jika berhasil, maka project kita akan dijalankan di dalam localhost menggunakan port 8000. Kita bisa membuaknya di dalam browser dengan http://localhost:8000.

Langkah 3 - Menjalankan Storage Link

Karena kita akan belajar melakukan upload gambar, maka kita perlu menjalankan perintah artisan storage:link.

Perintah tersebut digunakan untuk membuatkan link dari folder storage ke dalam folder public di Laravel. Silahkan jalankan perintah berikut ini di dalam terminal/CMD :

php artisan storage:link
