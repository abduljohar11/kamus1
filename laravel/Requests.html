Requests

  • Beberapa List Request

Laravel memiliki class Illuminate\Http\Request yang digunakan untuk menerima Http request yang dikirimkan, seperti input, cookies, files dan lain sebagainya.

Berikut ini contoh sederhana penggunaan request untuk mendapatkan data yang dikirimkan oleh user.

<?php
 
namespace App\Http\Controllers;
 
use Illuminate\Http\Request;
 
class UserController extends Controller
{
    /**
     * Store a new user.
     *
     * @param  \Illuminate\Http\Request  $request
     * @return \Illuminate\Http\Response
     */
    public function store(Request $request)
    {
        $name = $request->input('name');
 
        //
    }
}

Di dalam kode di atas, pertama kita import class Http request.

use Illuminate\Http\Request;

Kemudian di dalam method store kita memberikan parameter Request $request. Tujuaanya agar kita bisa menggunakan request di dalam method store.

Dan di dalamnya kita buat variable dengan nama $name yang berisi request input dengan nama name yang dikirimkan oleh user.

$name = $request->input('name');

Beberapa List Request

Berikut ini beberapa request yang harus teman-teman tahu.

Request Description
$request->all() Mengambil semua request yang dikirim oleh user.
$request->input('name') Mengambil request input bernama name yang dikirim oleh user.
$request->query('keywords') Mengambil request query bernama keywords yang ada di dalam URL.
$request->only(['username', 'password']); Mengambil hanya data username dan password yang dikirim oleh user.
$request->except(['credit_card']); Mengambil kecuali data credit_card yang dikirim oleh user.
$request->file('photo'); Mengambil request file bernama photo yang dikirim oleh user.
$request->cookie('name'); Mengambil request cookies bernama name dari browser.
$request->date('birthday'); Mengambil request date bernama birthday yang dikirim oleh user.
