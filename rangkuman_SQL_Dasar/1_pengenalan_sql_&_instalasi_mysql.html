Pengenalan SQL & Instalasi MySQL

SQL (Structured Query Language) adalah bahasa standar yang digunakan untuk mengelola data di dalam database relasional.

Analogi Dunia Nyata:
Bayangkan SQL adalah Bahasa Indonesia yang kamu gunakan untuk memesan makanan di warung. Pelayan (MySQL) akan mengerti pesananmu jika kamu menggunakan kalimat yang benar, seperti: "Tolong ambilkan (SELECT) semua ( * ) lauk dari (FROM) etalase (tabel_makanan)."

Apa Itu SQL?

Dengan SQL, kita memiliki kendali penuh atas data:

  • Membuat laci penyimpanan (Database/Tabel).
  • Menyimpan informasi baru (Insert).
  • Mengubah data yang salah (Update).
  • Mengambil data secara spesifik (Query).

Kenapa Harus MySQL?

MySQL sangat populer karena performanya stabil dan gratis. Dalam ekosistem Laravel, MySQL adalah pilihan database yang paling umum digunakan oleh pengembang.

Langkah 1 - Instalasi

Bagi pengguna Windows, cara termudah adalah menggunakan XAMPP:

  1. Download XAMPP di situs resmi Apache Friends.
  2. Buka XAMPP Control Panel dan aktifkan modul MySQL.
  3. Akses phpMyAdmin melalui browser di alamat http://localhost/phpmyadmin.

Contoh Penerapan: Query Pertama

Mari kita simulasikan pembuatan database sederhana untuk aplikasi sekolah:

-- 1. Membuat database sekolah
CREATE DATABASE belajar_sql;

-- 2. Menggunakan database tersebut
USE belajar_sql;

-- 3. Membuat tabel untuk menyimpan nama siswa
CREATE TABLE siswa (
  id INT AUTO_INCREMENT PRIMARY KEY,
  nama VARCHAR(50),
  umur INT
);

-- 4. Memasukkan data contoh
INSERT INTO siswa (nama, umur) VALUES ('Ali', 18), ('Budi', 17);

-- 5. Menampilkan hasilnya
SELECT * FROM siswa;

Kesimpulan

SQL adalah fondasi penting untuk pengelolaan data. Memahami perintah dasar seperti CREATE, INSERT, dan SELECT akan sangat membantumu saat nanti belajar fitur Migrations di Laravel 12.
