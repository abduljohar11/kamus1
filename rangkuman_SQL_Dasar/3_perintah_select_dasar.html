Perintah SELECT Dasar

Perintah SELECT adalah cara kita meminta data dari database untuk ditampilkan.

Analogi Memesan Makanan:
Bayangkan kamu berada di restoran (Database). Perintah SELECT adalah saat kamu berkata kepada pelayan: "Saya mau (SELECT) Nasi Goreng (Kolom) dari menu Utama (Tabel)."

Cara Mengambil Data

Gunakan simbol bintang (*) untuk mengambil semua kolom, atau sebutkan nama kolomnya untuk hasil yang lebih spesifik.

-- 1. Mengambil semua data dari tabel siswa
SELECT * FROM siswa;

-- 2. Hanya mengambil kolom nama dan kelas
SELECT nama, kelas FROM siswa;

-- 3. Memberi nama panggilan (Alias) agar lebih rapi
SELECT nama AS "Nama Siswa", email AS "Alamat Email" FROM siswa;

Kenapa Memilih Kolom Itu Penting?

Dalam aplikasi nyata (seperti Buku Guru), jangan selalu menggunakan SELECT *. Mengambil hanya data yang dibutuhkan (seperti Nama dan Nilai saja) akan membuat website kamu jauh lebih cepat dan hemat memori.
