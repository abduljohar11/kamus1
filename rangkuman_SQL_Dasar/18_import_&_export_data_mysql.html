IMPORT & EXPORT

Proses memindahkan data dari database ke file fisik (Export) atau sebaliknya (Import).

Analogi: Ibarat Pindahan Rumah. Export adalah memasukkan semua barang ke dalam kardus (file .sql), dan Import adalah menata kembali barang-barang tersebut di rumah baru.

Langkah Penting

Gunakan terminal atau phpMyAdmin untuk melakukan backup rutin aplikasi "Buku Guru" kamu:

# Export lewat terminal (Backup)
mysqldump -u root -p nama_database > backup_data.sql

# Import lewat terminal (Restore)
mysql -u root -p nama_database < backup_data.sql
