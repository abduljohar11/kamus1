Struktur Database & Tipe Data

Apa Itu Struktur Database?

Struktur database adalah cara kita mengatur data agar rapi dan mudah diolah.

Analogi Lemari Kabinet:
Bayangkan database adalah sebuah Lemari Kabinet besar.
  • Database: Lemarinya.
  • Tabel: Laci-laci di dalam lemari (misal: laci "Siswa", laci "Guru").
  • Kolom: Kategori informasi di dalam map (Nama, Alamat).
  • Baris: Satu map lengkap milik satu orang.

Tipe Data Populer

Setiap kolom wajib memiliki tipe data agar MySQL tahu cara memprosesnya.

  • INT: Angka bulat (ID, Umur).
  • VARCHAR(n): Teks pendek dinamis (Nama, Email).
  • TEXT: Teks sangat panjang (Biografi, Alamat Lengkap).
  • DATE/DATETIME: Tanggal dan waktu.
Hubungan dengan Laravel 12:
Saat membuat tabel di Laravel menggunakan Migrations, kamu akan menggunakan perintah seperti $table->string('nama') untuk VARCHAR atau $table->integer('umur') untuk INT.

Contoh Penerapan Nyata

-- Membuat tabel siswa dengan tipe data yang tepat
CREATE TABLE siswa (
  id INT PRIMARY KEY AUTO_INCREMENT, -- Angka unik otomatis
  nama VARCHAR(100),                -- Teks nama max 100 karakter
  email VARCHAR(50),                 -- Teks email
  bio TEXT,                          -- Catatan panjang
  tgl_lahir DATE                     -- Format YYYY-MM-DD
);
