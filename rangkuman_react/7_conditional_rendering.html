Conditional Rendering

Pada artikel kali ini, kita akan membahas tentang Conditional Rendering di React. Dengan teknik ini, kita bisa menampilkan elemen atau komponen secara dinamis berdasarkan kondisi tertentu.

Bayangkan sebuah Lampu Lalu Lintas. Aturannya sederhana: JIKA lampu hijau, maka kendaraan jalan. JIKA lampu merah, maka kendaraan berhenti. Tampilan di jalan raya (UI) berubah total hanya berdasarkan status (state) warna lampu tersebut.

Apa Itu Conditional Rendering?

Conditional Rendering adalah teknik untuk menentukan elemen mana yang akan dirender oleh React berdasarkan logika tertentu. Jika kondisi bernilai benar (true), maka elemen akan ditampilkan, sebaliknya jika salah (false) elemen tidak ditampilkan atau menampilkan alternatif lain.

Di React, kita bisa menggunakan beberapa cara:

  • if/else: Digunakan di dalam tubuh fungsi sebelum return.
  • Operator Ternary (? :): Cocok untuk kondisi singkat di dalam JSX.
  • Logical &&: Digunakan jika hanya ingin menampilkan sesuatu jika benar saja.

Menggunakan if/else

Cara ini paling mudah dipahami, kita melakukan pengecekan logika sebelum komponen mengembalikan (return) JSX.

function Greeting({ isLoggedIn }) {
  if (isLoggedIn) {
    return <h1>Selamat datang kembali!</h1>;
  }
  
  return <h1>Silakan login terlebih dahulu.</h1>;
}

Menggunakan Operator Ternary

Ini adalah cara paling populer karena kita bisa menyisipkan logika langsung di tengah-tengah kode JSX.

function App() {
  const isLoggedIn = true;

  return (
    <div>
      {isLoggedIn ? <p>Sudah Login</p> : <p>Belum Login</p>}
    </div>
  );
}

Menggunakan Operator &&

Terkadang kita hanya ingin menampilkan elemen jika kondisi terpenuhi, dan tidak ingin menampilkan apa pun jika gagal.

function Notification({ unreadCount }) {
  return (
    <div>
      <h1>Halo!</h1>
      {unreadCount > 0 && (
        <p>Kamu memiliki {unreadCount} pesan belum dibaca.</p>
      )}
    </div>
  );
}

Conditional Rendering dengan State

Inilah kekuatan asli React. UI akan berubah secara otomatis saat kita memperbarui state.

import { useState } from "react";

function AuthButton() {
  const [isLoggedIn, setIsLoggedIn] = useState(false);

  return (
    <div>
      <h1>{isLoggedIn ? "Selamat Datang" : "Silakan Login"}</h1>
      <button onClick={() => setIsLoggedIn(!isLoggedIn)}>
        {isLoggedIn ? "Logout" : "Login"}
      </button>
    </div>
  );
}

Kesimpulan

Pada artikel ini, kita sudah belajar tentang Conditional Rendering. Kita bisa menggunakan if/else untuk logika kompleks, operator ternary untuk pilihan dua arah, dan && untuk tampilan opsional. Teknik ini membuat aplikasi kita terasa cerdas karena mampu menyesuaikan diri dengan data.
