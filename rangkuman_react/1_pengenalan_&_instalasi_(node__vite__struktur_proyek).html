Pengenalan & Instalasi (Node, Vite, Struktur Proyek)

Semoga dengan seri ini, kita bisa memahami pondasi React dengan lebih kuat dan siap untuk melangkah ke tahap berikutnya.

Apa Itu React?

React adalah sebuah library JavaScript yang digunakan untuk membangun antarmuka pengguna (User Interface). React pertama kali dibuat oleh Facebook (sekarang Meta) dan kini menjadi salah satu library paling populer dalam pengembangan web modern.

[Image of React component-based architecture diagram]

Dengan React, kita bisa membuat komponen-komponen UI yang dapat digunakan kembali (reusable), mengelola data dengan mudah, dan membangun aplikasi single-page (SPA) yang cepat dan interaktif.

Jika diibaratkan:

  • HTML bertugas membangun struktur halaman.
  • CSS bertugas mengatur tampilan.
  • React hadir untuk mengatur bagaimana tampilan (UI) berubah secara dinamis ketika data berubah.

Kenapa Harus Belajar React?

  • Populer & Banyak Digunakan: Hampir semua perusahaan teknologi besar menggunakan React.
  • Komunitas Besar: Banyak dokumentasi, tutorial, dan dukungan dari komunitas global.
  • Ekosistem Luas: Didukung oleh banyak library pendukung (Router, State Management, dll).
  • Reusable Component: Membuat kode lebih terstruktur, rapi, dan mudah dipelihara.

Langkah 1 - Installasi Node.js

React membutuhkan Node.js untuk menjalankan server pengembangan dan mengelola dependensi (package). Silakan teman-teman download Node.js versi LTS di situs resmi:

https://nodejs.org/en/download

Setelah selesai, pastikan instalasi berhasil dengan menjalankan perintah berikut di terminal/CMD:

node -v
npm -v

Langkah 2 - Membuat Proyek React dengan Vite

Kita akan menggunakan Vite karena jauh lebih cepat dan ringan dibandingkan pendahulunya (Create React App).

Buka terminal, masuk ke folder tujuan, lalu jalankan perintah berikut:

npm create vite@latest belajar-react-dasar -- --template react

Setelah perintah di atas selesai, ikuti langkah navigasi berikut:

# Masuk ke folder proyek
cd belajar-react-dasar

# Install dependensi yang dibutuhkan
npm install

# Jalankan server pengembangan
npm run dev

Jika berhasil, buka browser lalu akses http://localhost:5173. Proyek React pertama kita sudah berjalan!

Struktur Folder Proyek React (Vite)

Memahami struktur folder awal sangatlah penting sebelum kita mulai menulis kode:

  • node_modules/ : Berisi semua library/package yang diinstall melalui NPM.
  • public/ : Tempat menyimpan aset statis seperti favicon atau logo.
  • src/ : Folder utama tempat kita menulis kode komponen React (JSX).
  • index.html : File HTML utama tempat aplikasi React kita di-mount.
  • package.json : Daftar library dan script proyek.
