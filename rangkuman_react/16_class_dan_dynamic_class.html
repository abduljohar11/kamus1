Class dan Dynamic Class

Materi ini sangat krusial karena dalam pembuatan aplikasi nyata, elemen UI sering kali harus berubah bentuk, warna, atau posisi berdasarkan status tertentu (misal: tombol yang menyala saat aktif).

Bayangkan sebuah Jaket Bolak-Balik. Sisi luar berwarna biru (class: blue-side) dan sisi dalam berwarna merah (class: red-side). Anda adalah React yang memutuskan sisi mana yang dipakai berdasarkan cuaca atau suasana hati (state). Jaketnya tetap sama, tapi "penampilannya" (class) berubah secara dinamis.

Menambahkan Class Biasa

Ingat, di JSX kita dilarang menggunakan kata class karena merupakan kata kunci cadangan JavaScript untuk membuat objek. Kita wajib menggunakan className.

File CSS (App.css):

.btn { padding: 10px 20px; border-radius: 5px; }
.active { background-color: green; color: white; }
.inactive { background-color: gray; color: black; }

Menambahkan Class Secara Dinamis

Cara termudah adalah menggunakan ternary operator di dalam atribut className.

import { useState } from 'react';
import './App.css';

function App() {
  const [isActive, setIsActive] = useState(false);

  return (
    <button 
      className={isActive ? "btn active" : "btn inactive"}
      onClick={() => setIsActive(!isActive)}
    >
      {isActive ? "AKTIF" : "NON-AKTIF"}
    </button>
  );
}

Menggunakan Template Literal

Jika Anda memiliki banyak class statis dan hanya satu atau dua yang dinamis, gunakan backtick (`).

function Box({ isDark }) {
  return (
    <div className={`box-layout border shadow ${isDark ? 'bg-black' : 'bg-white'}`}>
       Konten Box
    </div>
  );
}

Menggunakan Library clsx (Sangat Disarankan)

Untuk pengelolaan class yang sangat kompleks (misal pada Tailwind CSS), library clsx atau classnames adalah standar industri.

import clsx from 'clsx';

function StatusBadge({ status }) {
  return (
    <span className={clsx('badge-base', {
      'bg-red-500': status === 'error',
      'bg-green-500': status === 'success',
      'bg-yellow-500': status === 'pending',
    })}>
      {status}
    </span>
  );
}
