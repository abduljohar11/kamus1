List & Keys

Pada artikel kali ini, kita akan membahas tentang List & Keys di React. Fitur ini sangat penting karena hampir semua aplikasi modern menampilkan data dalam bentuk daftar, seperti feed sosial media atau daftar produk belanja.

Apa Itu List?

List di React digunakan untuk menampilkan data yang berbentuk array secara berulang. Alih-alih menulis elemen satu per satu, kita menggunakan fungsi map() bawaan JavaScript untuk melakukan iterasi.

Menampilkan List dengan map()

Mari kita buat contoh daftar nama sederhana.

function NameList() {
  const names = ["Budi", "Siti", "Andi", "Raza"];

  return (
    <ul>
      {names.map((name, index) => (
        <li key={index}>{name}</li>
      ))}
    </ul>
  );
}

Bayangkan sebuah Gantungan Kunci di Hotel. Setiap kunci memiliki nomor kamar (Key). Tanpa nomor tersebut, resepsionis akan kebingungan saat tamu ingin menaruh atau mengambil kunci tertentu. Di React, Key adalah nomor kamar tersebut yang membantu React mengenali elemen mana yang ditambah, diubah, atau dihapus dalam sebuah daftar panjang.

Kenapa Keys Sangat Penting?

Key membantu React mengidentifikasi item mana yang telah berubah. Tanpa key yang unik, React akan me-render ulang seluruh list setiap kali ada satu perubahan kecil, yang tentu saja akan memperlambat performa aplikasi Anda.

Menampilkan List dengan Object

Dalam proyek nyata, data biasanya berbentuk array of objects yang memiliki ID unik dari database.

function UserList() {
  const users = [
    { id: 101, name: "Budi", role: "Admin" },
    { id: 102, name: "Siti", role: "Editor" },
    { id: 103, name: "Andi", role: "User" }
  ];

  return (
    <ul>
      {users.map((user) => (
        <li key={user.id}>
          {user.name} - {user.role}
        </li>
      ))}
    </ul>
  );
}

List dengan Component Terpisah

Agar kode lebih modular, kita bisa memisahkan tampilan item list ke dalam komponen sendiri.

function UserItem({ name, role }) {
  return <li>{name} bertugas sebagai {role}</li>;
}

function App() {
  const users = [...]; // data dari atas

  return (
    <ul>
      {users.map((user) => (
        <UserItem key={user.id} name={user.name} role={user.role} />
      ))}
    </ul>
  );
}

Kesimpulan

Pada artikel ini, kita sudah belajar cara merender data dinamis menggunakan map() dan memahami peran krusial Key untuk menjaga performa aplikasi tetap cepat. Ingat: selalu gunakan ID unik jika tersedia, dan gunakan index hanya sebagai pilihan terakhir.
