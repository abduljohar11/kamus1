Components

Pada artikel kali ini, kita akan membahas Components. Konsep ini sangat penting karena hampir seluruh aplikasi React dibangun dengan komponen.

Dengan memahami components, kita akan lebih mudah menyusun aplikasi menjadi potongan-potongan kecil yang bisa digunakan kembali (reusable).

Apa Itu Components?

Component adalah bagian kecil dari UI yang bisa kita buat, gunakan kembali, dan gabungkan untuk membentuk aplikasi yang lebih besar.

Jika diibaratkan rumah adalah sebuah website, maka component adalah bagian-bagian penyusunnya seperti pintu, jendela, atau meja. Semua bagian itu bisa dipakai ulang di ruangan lain secara independen.

React mendukung dua jenis komponen:

  • Functional Component: Menggunakan fungsi JavaScript biasa (paling populer dan modern).
  • Class Component: Menggunakan syntax Class ES6 (sudah mulai ditinggalkan namun masih ada di proyek lama).

Membuat Functional Component

Silakan buka file src/App.jsx dan ubah kodenya menjadi seperti berikut:

// Ini adalah sebuah Component bernama Welcome
function Welcome() {
  return <h1>Selamat Datang di React!</h1>;
}

// Ini adalah Component Utama
function App() {
  return (
    <div>
      <Welcome />
      <p>Belajar komponen itu sangat mudah.</p>
    </div>
  );
}

export default App;

Component Bisa Dipakai Ulang

Kelebihan utama dari component adalah kemampuannya untuk dipanggil berkali-kali tanpa menulis ulang struktur kodenya.

function App() {
  return (
    <div>
      <Welcome />
      <Welcome />
      <Welcome />
    </div>
  );
}

Aturan Penamaan Component

Di React, ada satu aturan wajib: nama component harus diawali huruf besar.

  • <welcome />: Dianggap oleh React sebagai elemen HTML biasa (yang tidak ada di standar HTML).
  • <Welcome />: Dianggap sebagai component React buatan kita sendiri.

Menyimpan Component di File Terpisah

Supaya kode lebih rapi, biasanya setiap component disimpan di file terpisah. Mari kita coba!

1. Buat folder baru src/components, lalu buat file bernama Header.jsx:

function Header() {
  return (
    <header>
       <h1>Ini adalah Header Aplikasi</h1>
    </header>
  );
}

export default Header;

2. Gunakan di dalam file src/App.jsx dengan cara meng-import-nya:

import Header from './components/Header';

function App() {
  return (
    <div>
      <Header />
      <main>Konten Utama Aplikasi</main>
    </div>
  );
}

export default App;
