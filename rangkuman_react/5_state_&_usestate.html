State & useState

Pada artikel kali ini, kita akan membahas tentang State dan useState. Konsep ini sangat penting karena dengan state, kita bisa menyimpan dan mengelola data yang interaktif di dalam sebuah component.

Apa Itu State?

State adalah data internal pada sebuah component yang bisa berubah seiring interaksi pengguna. Berbeda dengan props yang dikirim dari luar (induk), state dikelola langsung oleh component itu sendiri.

Jika state berubah, maka React akan otomatis melakukan re-render UI agar tampilan selalu sinkron dengan data terbaru.

Apa Itu useState?

useState adalah Hook bawaan React yang digunakan untuk menambahkan state ke dalam functional component.

Fungsi ini mengembalikan sebuah array dengan 2 nilai:

  • state: Nilai data saat ini (current value).
  • setState: Fungsi untuk memperbarui nilai state.

Sintaks dasar:

const [name, setName] = useState("Nilai Awal");

Membuat State Dasar

Mari kita buat contoh sederhana untuk mengubah teks saat tombol ditekan.

import { useState } from "react";

function App() {
  const [message, setMessage] = useState("Halo, React!");

  return (
    <div>
      <h1>{message}</h1>
      <button onClick={() => setMessage("State telah berubah!")}>
        Ubah Teks
      </button>
    </div>
  );
}

export default App;

State dengan Angka (Counter)

Contoh paling umum dari penggunaan state adalah membuat sebuah counter (penghitung angka).

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Angka saat ini: {count}</p>
      <button onClick={() => setCount(count + 1)}>Tambah</button>
      <button onClick={() => setCount(count - 1)}>Kurangi</button>
    </div>
  );
}

State Menyimpan Array

State tidak hanya bisa menyimpan teks atau angka, tapi juga data kompleks seperti array atau object.

function TodoList() {
  const [todos, setTodos] = useState(["Belajar React", "Belajar State"]);

  const addTodo = () => {
    // Gunakan Spread Operator untuk menambah item ke array
    setTodos([...todos, "Item Baru"]);
  };

  return (
    <div>
      <button onClick={addTodo}>Tambah Todo</button>
      <ul>
        {todos.map((todo, index) => (
          <li key={index}>{todo}</li>
        ))}
      </ul>
    </div>
  );
}
