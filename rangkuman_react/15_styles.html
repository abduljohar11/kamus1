Styles

Styling adalah elemen vital dalam membangun aplikasi yang user-friendly. Di React, cara kita memberikan gaya sedikit berbeda dari HTML tradisional karena kita bekerja dalam ekosistem JavaScript.

Bayangkan Anda sedang Mengecat Rumah. Inline Style seperti mencelupkan kuas dan mengecat satu bata secara langsung. CSS Eksternal seperti memiliki buku panduan warna di luar rumah. Sedangkan CSS Modules adalah cat khusus yang hanya menempel pada satu bata spesifik tanpa merembet ke bata lain di rumah tetangga.

Inline Style

Di React, inline style ditulis menggunakan objek JavaScript. Properti CSS yang biasanya menggunakan tanda hubung (dash) diubah menjadi camelCase.

function App() {
  const headerStyle = {
    color: "white",
    backgroundColor: "DodgerBlue",
    padding: "20px",
    fontFamily: "Arial"
  };

  return <h1 style={headerStyle}>Halo React!</h1>;
}

File CSS Eksternal

Ini adalah cara klasik. Kita membuat file CSS dan mengimpornya langsung ke dalam file komponen menggunakan import.

Buat file App.css:

.container {
  padding: 2rem;
  background-color: #f1f5f9;
  border-radius: 8px;
}

Lalu gunakan di App.jsx:

import './App.css';

function App() {
  return <div className="container">Konten bergaya CSS Eksternal</div>;
}

CSS Modules

Masalah CSS tradisional adalah lingkup globalnya. CSS Modules memecahkan ini dengan memberikan nama unik ke setiap class secara otomatis sehingga tidak terjadi bentrokan style antar komponen.

Buat file Button.module.css:

.btnBlue {
  background-color: blue;
  color: white;
}

Gunakan di komponen:

import styles from './Button.module.css';

function MyButton() {
  return <button className={styles.btnBlue}>Klik Biru</button>;
}

Menggunakan Library Styling

Dua metode paling populer saat ini:

  • Tailwind CSS: Menggunakan utility classes langsung di JSX (sangat cepat).
  • Styled Components: Menulis CSS murni di dalam variabel JavaScript.

Contoh dengan Styled Components:

import styled from 'styled-components';

const Wrapper = styled.div`
  background: palevioletred;
  padding: 10px;
  border-radius: 5px;
`;

function App() {
  return <Wrapper>Saya adalah komponen bergaya!</Wrapper>;
}

Kesimpulan

Setiap cara styling memiliki keunggulannya masing-masing. Untuk proyek kecil, CSS biasa sudah cukup. Namun untuk aplikasi skala besar, CSS Modules atau Tailwind CSS sangat disarankan untuk menjaga kode tetap rapi dan terhindar dari konflik style.

Di artikel berikutnya, kita akan membahas tentang Class dan Dynamic Class, yaitu teknik mengatur class CSS secara otomatis berdasarkan state atau kondisi tertentu!
