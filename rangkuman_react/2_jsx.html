Mengenal JSX

Pada artikel kali ini, kita akan membahas salah satu konsep paling penting dalam React, yaitu JSX. Konsep ini wajib kita pahami karena hampir semua kode React ditulis dengan menggunakan JSX.

Apa Itu JSX?

JSX (JavaScript XML) adalah ekstensi sintaks JavaScript yang memungkinkan kita menulis kode HTML di dalam JavaScript. JSX bukanlah string atau template engine, melainkan sebuah sintaks khusus yang akan dikompilasi menjadi fungsi JavaScript murni.

Contoh penulisan JSX:

const element = <h1>Hello World</h1>;

Dibalik layar, React (melalui compiler seperti Babel atau SWC) akan mengubahnya menjadi fungsi React.createElement():

const element = React.createElement('h1', null, 'Hello World');

Kenapa Harus Menggunakan JSX?

  • Mudah Dibaca: Struktur kode lebih rapi karena menyerupai HTML.
  • Deklaratif: Kita cukup mendeskripsikan bagaimana UI seharusnya terlihat.
  • Keamanan: JSX mencegah serangan XSS secara default dengan membersihkan nilai yang disisipkan sebelum dirender.

Menulis JSX Dasar

Silakan buka file src/App.jsx dan ubah kodenya menjadi seperti berikut:

function App() {
  return (
    <div>
      <h1>Hello React</h1>
      <p>Belajar JSX itu menyenangkan!</p>
    </div>
  );
}

export default App;

Menyisipkan Ekspresi JavaScript

JSX memungkinkan kita menulis ekspresi JavaScript (variabel, fungsi, operasi matematika) di dalam tanda kurung kurawal { }.

function App() {
  const name = "Santri Koding";
  const age = 20;

  return (
    <div>
      <h1>Halo, {name}!</h1>
      <p>Umur saya tahun depan adalah: {age + 1} tahun</p>
    </div>
  );
}

export default App;

Aturan Dasar JSX

Ada beberapa aturan yang wajib diikuti saat menulis JSX:

1. Hanya boleh ada satu elemen induk (Parent Element)

// âœ… Benar: Dibungkus dalam satu div atau Fragment
return (
  <>
    <h1>Hello</h1>
    <p>World</p>
  </>
);

2. Gunakan className bukan class

Karena class adalah kata kunci cadangan di JavaScript, kita menggunakan className untuk memberikan kelas CSS.

<h1 className="text-bold text-blue">Judul Artikel</h1>

3. Gunakan camelCase untuk atribut

Atribut seperti onclick berubah menjadi onClick, dan tabindex menjadi tabIndex.

<button onClick={() => alert("Halo!")}>Klik Saya</button>

Kesimpulan

JSX adalah jembatan yang memudahkan kita membangun UI di React dengan menggabungkan kekuatan JavaScript dan struktur HTML. Dengan memahami aturan dasarnya, kita sudah siap untuk membuat komponen yang lebih kompleks.

Di artikel berikutnya, kita akan belajar tentang Components, yaitu pondasi utama dalam membangun aplikasi React yang modular.
