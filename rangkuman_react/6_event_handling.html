Event Handling

Pada artikel kali ini, kita akan membahas tentang Event Handling di React. Dengan event handling, kita bisa merespons interaksi pengguna seperti klik tombol, mengetik di input, mengirim form, dan berbagai aksi lainnya.

Bayangkan sebuah Bel Rumah. Tombol bel adalah elemen UI, "Tamu yang menekan tombol" adalah Event (kejadian), dan "Suara bel yang berbunyi" adalah Handler (respon yang kita siapkan). Tanpa handler, bel ditekan ribuan kali pun tidak akan terjadi apa-apa.

Apa Itu Event Handling?

Event Handling adalah cara untuk menangani sebuah aksi atau interaksi yang dilakukan pengguna terhadap elemen di UI. React menggunakan sistem Synthetic Events yang membungkus event asli browser agar berjalan konsisten di semua browser.

Aturan penting di React:

  • Menggunakan camelCase (contoh: onClick, bukan onclick).
  • Mengirim fungsi sebagai handler, bukan string.

Event onClick

Digunakan untuk menangani aksi klik. Mari kita buat fungsi handleClick.

function App() {
  const handleClick = () => {
    alert("Tombol telah diklik!");
  };

  return (
    <button onClick={handleClick}>
      Klik Saya
    </button>
  );
}

Event onChange (Input)

Event onChange sangat penting untuk menangkap apa yang diketikkan pengguna di dalam input secara real-time.

import { useState } from "react";

function App() {
  const [name, setName] = useState("");

  return (
    <div>
      <input 
        type="text" 
        placeholder="Ketik nama..." 
        onChange={(e) => setName(e.target.value)} 
      />
      <p>Halo, {name}!</p>
    </div>
  );
}

Event onSubmit (Form)

Digunakan pada elemen <form>. Kita wajib menggunakan event.preventDefault() agar browser tidak me-refresh halaman saat submit.

function App() {
  const handleSubmit = (event) => {
    event.preventDefault();
    alert("Data berhasil dikirim!");
  };

  return (
    <form onSubmit={handleSubmit}>
      <input type="email" placeholder="Email" />
      <button type="submit">Kirim</button>
    </form>
  );
}

Event dengan Parameter

Jika ingin mengirim data ke dalam fungsi handler, kita harus membungkusnya dalam arrow function.

function App() {
  const sayHello = (name) => {
    alert("Halo " + name);
  };

  return (
    <button onClick={() => sayHello("Santri Koding")}>
      Sapa Saya
    </button>
  );
}
