Fetching Data (API)

Aplikasi React yang statis tidak akan berguna banyak. Di materi ini, kita akan belajar cara mengambil data dari server (API) dan menampilkannya ke dalam UI kita secara dinamis.

ðŸ’¡ Analogi

Bayangkan Anda di sebuah Restoran. React adalah Meja Makan Anda. Mengambil data dari API adalah seperti Pelayan yang pergi ke dapur (Server) untuk mengambil makanan (Data) dan membawanya ke meja Anda. Anda harus menunggu beberapa saat (Loading) sebelum makanan sampai.

Menggunakan Fetch API

React biasanya menggunakan useEffect untuk memicu pengambilan data saat komponen pertama kali muncul. Kita akan menggunakan JSONPlaceholder sebagai sumber data gratis.

import { useState, useEffect } from 'react';

function UserList() {
  const [users, setUsers] = useState([]);
  const [isLoading, setIsLoading] = useState(true);

  useEffect(() => {
    // Memulai pengambilan data
    fetch('https://jsonplaceholder.typicode.com/users')
      .then((response) => response.json())
      .then((data) => {
        setUsers(data);
        setIsLoading(false); // Matikan loading setelah data sampai
      })
      .catch((error) => console.error("Gagal ambil data:", error));
  }, []); // [] memastikan hanya berjalan sekali saat mount

  if (isLoading) return <p>Sedang memuat data...</p>;

  return (
    <ul>
      {users.map((user) => (
        <li key={user.id}>{user.name}</li>
      ))}
    </ul>
  );
}

Menggunakan Async/Await

Penulisan async/await sering dianggap lebih rapi dan mudah dibaca dibandingkan .then().

useEffect(() => {
  const fetchData = async () => {
    try {
      const response = await fetch('https://jsonplaceholder.typicode.com/users');
      const data = await response.json();
      setUsers(data);
    } catch (error) {
      console.log("Terjadi kesalahan:", error);
    } finally {
      setIsLoading(false);
    }
  };

  fetchData();
}, []);

Library Pihak Ketiga: Axios

Banyak developer profesional lebih suka menggunakan Axios karena fitur-fiturnya lebih lengkap (otomatis parse JSON, timeout, interceptors, dll).

npm install axios

Contoh penggunaan Axios:

import axios from 'axios';

useEffect(() => {
  axios.get('https://jsonplaceholder.typicode.com/users')
    .then(res => setUsers(res.data))
    .catch(err => console.log(err));
}, []);

Kesimpulan

Kemampuan mengambil data adalah syarat mutlak bagi seorang React Developer. Dengan menggabungkan useState untuk menyimpan data dan useEffect untuk mengambilnya, Anda sudah bisa membuat aplikasi yang benar-benar hidup dan dinamis.
