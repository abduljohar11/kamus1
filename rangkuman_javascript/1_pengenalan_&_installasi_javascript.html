Pengenalan & Installasi JavaScript

Apa Itu JavaScript?

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat website menjadi lebih interaktif. Bersama dengan HTML dan CSS, JavaScript membentuk pondasi utama dari pengembangan web modern.

ðŸ’¡ Analogi

Bayangkan sebuah Mobil:

  • HTML adalah rangka dan bodi mobil (Struktur).
  • CSS adalah cat dan desain interior (Tampilan).
  • JavaScript adalah mesin, rem, dan sistem kelistrikannya. Tanpa JS, mobil hanyalah pajangan; dengan JS, mobil bisa berjalan, mengerem, dan menyalakan lampu (Interaksi).

JavaScript juga kini digunakan di luar browser, contohnya di sisi server menggunakan Node.js.

Kenapa Harus Belajar JavaScript?

  • Mudah Dipelajari: Cocok untuk pemula yang baru mulai belajar coding.
  • Fleksibel: Bisa digunakan untuk frontend, backend, hingga mobile app.
  • Populer: Banyak perusahaan membutuhkan developer JavaScript.
  • Banyak Framework: Seperti React, Vue, Angular, dan lainnya.

Bagaimana Cara Menjalankan JavaScript?

Ada dua cara paling umum:

  • Langsung dari Browser: Semua browser modern sudah menyediakan JavaScript Engine yang memungkinkan kita menjalankan JavaScript langsung.
  • Menggunakan Node.js: Node.js adalah runtime JavaScript di luar browser. Dengan Node.js, kita bisa menjalankan JavaScript dari terminal.

Menjalankan JavaScript di Browser

Untuk menjalankan JavaScript di browser ada 2 cara, yaitu:

Cara 1: Melalui Developer Tools (Console)

Buka browser (Chrome/Edge/Firefox). Tekan Ctrl + Shift + I atau klik kanan âž” "Inspect" âž” pilih tab Console. Ketik kode berikut:

console.log('Hello, World!');
alert('Belajar JS itu seru!');

Cara 2: Menjalankan File .html di Browser

Gunakan tag <script> di dalam file HTML Anda.

<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <title>Belajar JavaScript</title>
</head>
<body>
  <h1>Hello JavaScript</h1>

  <script>
    console.log('Hello dari file HTML!');
    document.querySelector('h1').innerText = 'Struktur HTML Diubah JS!';
  </script>
</body>
</html>

Menjalankan JavaScript dengan Node.js

Node.js memungkinkan kita menjalankan JavaScript di terminal tanpa perlu membuka browser.

Cara Install Node.js

Download versi LTS dari situs resmi: https://nodejs.org/

Cek Hasil Installasi

Ketikkan perintah berikut di terminal/CMD untuk memastikan Node.js sudah aktif:

node --version
npm --version

Menjalankan File JavaScript

Buat file bernama hello.js, lalu jalankan lewat terminal:

// hello.js
console.log('Hello dari Node.js!');
node hello.js
