Array

Apa Itu Array?

Array adalah sebuah struktur data di JavaScript yang digunakan untuk menyimpan banyak nilai dalam satu variabel. Jika variabel biasa ibarat sebuah kotak tunggal, maka array adalah kotak panjang yang memiliki banyak sekat di dalamnya.

Bayangkan sebuah Lemari Laci. Seluruh lemari tersebut adalah Array. Setiap laci memiliki nomor urut (Index) yang dimulai dari angka 0. Di dalam setiap laci, Anda bisa menyimpan barang (Data). Jika Anda ingin mengambil baju, Anda cukup ingat baju tersebut ada di laci nomor berapa.

Membuat Array di JavaScript

Cara paling umum dan direkomendasikan adalah menggunakan tanda kurung siku [].

// Cara 1: Menggunakan literal []
let buah = ["Apel", "Pisang", "Jeruk"];

// Cara 2: Menggunakan new Array()
let angka = new Array(1, 2, 3, 4, 5);

Mengakses & Mengubah Data

Data diakses menggunakan nomor index. Ingat, index selalu dimulai dari 0.

let buah = ["Apel", "Pisang", "Jeruk"];

console.log(buah[0]); // Apel
console.log(buah[1]); // Pisang

// Mengubah data
buah[2] = "Mangga"; 
console.log(buah); // ["Apel", "Pisang", "Mangga"]

Method Penting Array

JavaScript menyediakan banyak fungsi (method) bawaan untuk mempermudah kita mengolah data array:

  • push(): Menambah item di akhir.
  • pop(): Menghapus item terakhir.
  • shift(): Menghapus item pertama.
  • unshift(): Menambah item di awal.
  • length: Properti untuk mengecek jumlah data.
let hewan = ["Kucing", "Anjing"];

hewan.push("Kelinci"); // ["Kucing", "Anjing", "Kelinci"]
hewan.pop();           // ["Kucing", "Anjing"]
console.log(hewan.length); // 2

Iterasi & Transformasi

Untuk mengolah setiap data di dalam array secara otomatis, kita bisa menggunakan method tingkat tinggi (Higher Order Functions):

1. map()

Digunakan untuk membuat array baru berdasarkan proses tertentu dari array lama.

let angka = [1, 2, 3];
let kaliDua = angka.map(x => x * 2); // [2, 4, 6]

2. filter()

Menyaring data berdasarkan kondisi tertentu.

let nilai = [50, 80, 40, 90];
let lulus = nilai.filter(n => n >= 75); // [80, 90]

Kesimpulan

Array adalah jantung dari pengelolaan data di JavaScript. Dengan memahami cara menyimpan, mengakses, dan memanipulasi data melalui method-method di atas, Anda sudah siap untuk menangani data yang lebih kompleks.

Di artikel berikutnya, kita akan membahas tentang Object di JavaScript: cara menyimpan data yang memiliki identitas lebih spesifik!
