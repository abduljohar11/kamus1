Function

Apa Itu Function?

Function adalah sebuah blok kode yang dirancang untuk melakukan tugas tertentu dan bisa digunakan berulang-ulang tanpa perlu menulis ulang kodenya. Dengan function, aplikasi kita menjadi lebih terstruktur dan efisien.

ðŸ’¡ Analogi

Bayangkan sebuah Mesin Jus. Parameter adalah buah yang Anda masukkan (Input). Blok Kode di dalam fungsi adalah proses blender/memeras buah tersebut. Dan Return Value adalah segelas jus yang keluar dari mesin (Output). Anda cukup membuat mesinnya sekali, dan bisa menggunakannya kapan pun Anda butuh jus buah apa pun.

Membuat Function (Declaration)

Cara paling standar adalah menggunakan kata kunci function diikuti nama fungsinya.

function sapa() {
  console.log("Halo, selamat belajar JavaScript!");
}

// Cara memanggilnya:
sapa();

Function dengan Parameter

Kita bisa mengirimkan data ke dalam fungsi melalui parameter agar hasil fungsinya lebih dinamis.

function kirimPesan(nama, waktu) {
  console.log("Halo " + nama + ", Selamat " + waktu + "!");
}

kirimPesan("Budi", "Pagi");
kirimPesan("Siti", "Malam");

Function dengan Return Value

Function juga bisa mengembalikan hasil olahannya agar bisa disimpan ke dalam variabel atau digunakan oleh bagian lain dari program.

function tambah(a, b) {
  return a + b;
}

let hasil = tambah(10, 5);
console.log("Hasilnya adalah: " + hasil); // 15

Function Expression

Selain cara deklarasi di atas, fungsi juga bisa disimpan langsung ke dalam variabel. Ini disebut Function Expression.

const sapaDunia = function() {
  console.log("Halo Dunia!");
};

sapaDunia();
