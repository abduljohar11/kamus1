Tipe Data

Apa Itu Tipe Data?

Tipe data adalah klasifikasi jenis nilai yang bisa kita simpan dan operasikan di dalam sebuah variabel. JavaScript adalah bahasa yang bersifat dynamic typing, artinya satu variabel bisa berganti tipe data secara otomatis.

Komputer perlu tahu "label" apa yang sedang ia baca agar tidak salah menghitung; Anda tidak bisa menjumlahkan "Apel" dengan "Rp5000" secara matematika, bukan?

Tipe Data Primitive

Tipe data sederhana yang menyimpan nilai tunggal. Berikut macam-macamnya:

  • String: Representasi teks. Ditulis di dalam petik '...', "...", atau `...`.
  • Number: Angka bulat maupun desimal.
  • Boolean: Hanya bernilai true atau false.
  • Undefined: Variabel yang sudah ada tapi belum diberi isi.
  • Null: Representasi nilai kosong yang disengaja.
let nama = "SantriKoding"; // String
let umur = 25;             // Number
let isOnline = true;       // Boolean
let alamat;                // Undefined (karena kosong)
let data = null;           // Null (sengaja dikosongkan)

Tipe Data Non-Primitive

Tipe data kompleks yang bisa menampung banyak nilai sekaligus.

Object

Menyimpan data dengan pasangan key-value (seperti identitas lengkap).

let mahasiswa = {
  nama: "Riza",
  jurusan: "Informatika",
  lulus: false
};

Array

Menyimpan daftar nilai yang berurutan menggunakan index angka.

let buah = ["Apel", "Jeruk", "Mangga"];
console.log(buah[0]); // Output: Apel

Mengetahui Tipe Data (typeof)

Gunakan operator typeof untuk mengecek jenis data dari sebuah variabel secara real-time.

console.log(typeof "Halo");  // "string"
console.log(typeof 123);     // "number"
console.log(typeof true);    // "boolean"
console.log(typeof {});      // "object"

Catatan Penting: typeof null akan mengembalikan "object". Ini adalah anomali bersejarah di JavaScript, namun tetap dianggap sebagai tipe data primitive.

Kesimpulan

Memahami tipe data adalah langkah awal untuk menghindari error saat melakukan operasi logika atau matematika. Dengan mengetahui perbedaan Primitive dan Non-Primitive, Anda bisa mengelola memori aplikasi dengan lebih baik.
