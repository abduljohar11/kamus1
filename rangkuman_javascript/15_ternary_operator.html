Ternary Operator

Apa Itu Ternary Operator?

Ternary Operator adalah operator kondisional yang berbentuk satu baris, digunakan untuk menggantikan struktur if-else sederhana. Disebut "ternary" karena melibatkan tiga bagian utama: kondisi, hasil jika true, dan hasil jika false.

ðŸ’¡ Analogi

Bayangkan sebuah Gerbang Otomatis di parkiran mal. Kondisi: "Apakah saldo kartu cukup?". Jika YA, palu gerbang terbuka. Jika TIDAK, palu tetap tertutup. Semuanya terjadi dalam satu aksi cepat di satu titik gerbang tersebut.

Format umum penulisan Ternary Operator adalah sebagai berikut:

kondisi ? hasil_jika_true : hasil_jika_false;

Contoh Tanpa Ternary Operator

Biasanya jika kita ingin membuat percabangan sederhana untuk memberikan nilai ke variabel, kita menggunakan if-else:

let umur = 20;
let status;

if (umur >= 18) {
  status = "Dewasa";
} else {
  status = "Anak-anak";
}

console.log(status); // Output: Dewasa

Cukup panjang hanya untuk satu keputusan sederhana, bukan?

Contoh Dengan Ternary Operator

Dengan Ternary Operator, kita bisa meringkas kode di atas menjadi satu baris saja:

let umur = 20;
let status = (umur >= 18) ? "Dewasa" : "Anak-anak";

console.log(status); // Output: Dewasa

Lebih singkat, bersih, dan tetap mudah dibaca!

Cara Kerja Ternary Operator

  • Kondisi (sebelum tanda ?) akan dicek terlebih dahulu nilai kebenarannya.
  • Jika Benar (true), ekspresi di antara tanda ? dan : akan dieksekusi.
  • Jika Salah (false), ekspresi setelah tanda : akan dieksekusi.

Contoh Lain: Cek Kelulusan

let nilai = 75;
let hasil = (nilai >= 70) ? "Lulus" : "Tidak Lulus";

console.log(`Anda dinyatakan: ${hasil}`); // Output: Anda dinyatakan: Lulus

Ternary Operator Bertingkat (Nested)

Kita juga bisa membuat ternary operator bertingkat untuk menangani lebih dari dua kondisi, mirip dengan if-else if-else:

let skor = 85;
let grade = (skor >= 90) ? "A" 
          : (skor >= 80) ? "B" 
          : (skor >= 70) ? "C" 
          : "D";

console.log(grade); // Output: B

Tips: Gunakan ternary bertingkat dengan bijak. Jika kondisinya terlalu banyak, sebaiknya kembali menggunakan switch atau if-else agar kode tetap mudah dipahami oleh orang lain.

Kesimpulan

Ternary Operator di JavaScript adalah alat yang sangat berguna untuk membuat keputusan cepat dalam satu baris. Ini sangat sering digunakan dalam pengembangan **React** untuk menampilkan elemen UI secara kondisional.

Di artikel berikutnya, kita akan membahas materi yang sangat penting untuk modularitas kode, yaitu Module (Import & Export) di JavaScript!
