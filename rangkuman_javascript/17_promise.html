Promise

Apa Itu Promise di JavaScript?

Promise adalah sebuah objek yang mewakili keberhasilan atau kegagalan dari sebuah operasi asinkronus (proses yang membutuhkan waktu) di masa depan.

ðŸ’¡ Analogi

Bayangkan Anda memesan burger di Restoran Cepat Saji. Setelah membayar, Anda diberi alat Buzzer. Alat itu adalah Promise.
1. Pending: Buzzer diam, burger sedang dimasak.
2. Fulfilled: Buzzer bergetar, burger siap diambil (Berhasil).
3. Rejected: Pelayan bilang bahan habis, burger tidak jadi dibuat (Gagal).

Promise memiliki tiga kemungkinan status:

  • Pending: Status awal, proses sedang berjalan.
  • Fulfilled: Operasi selesai dengan sukses.
  • Rejected: Operasi gagal.

Cara Membuat Promise

Gunakan constructor new Promise yang menerima dua callback: resolve (sukses) dan reject (gagal).

const janjiNonton = new Promise((resolve, reject) => {
  const cuacaCerah = true;

  if (cuacaCerah) {
    resolve("Ayo berangkat nonton!");
  } else {
    reject("Yah, hujan. Batal nonton.");
  }
});

Cara Menggunakan Promise

Setelah Promise dibuat, kita menangani hasilnya menggunakan method .then() untuk sukses dan .catch() untuk error.

janjiNonton
  .then((hasil) => {
    console.log(hasil); // "Ayo berangkat nonton!"
  })
  .catch((error) => {
    console.log(error); // "Yah, hujan..."
  })
  .finally(() => {
    console.log("Proses selesai.");
  });

Promise dengan setTimeout (Simulasi API)

Biasanya Promise digunakan untuk proses yang memakan waktu, seperti mengambil data dari internet.

const ambilDataUser = new Promise((resolve) => {
  console.log("Mengambil data...");
  setTimeout(() => {
    resolve({ id: 1, nama: "Budi" });
  }, 2000); // Menunggu 2 detik
});

ambilDataUser.then(user => console.log(user));

Kesimpulan

Promise membantu kita mengelola kode yang berjalan di latar belakang tanpa menghentikan jalannya seluruh program. Ini adalah pondasi dasar sebelum kita mempelajari teknik yang lebih canggih lagi.
