Control Flow

Apa Itu Control Flow?

Control Flow adalah cara kita mengatur urutan eksekusi perintah di dalam program. Secara default, kode dibaca dari atas ke bawah, namun dengan control flow kita bisa membuat lompatan, percabangan, atau pengulangan.

Bayangkan Anda sedang Menyetir Mobil mengikuti GPS. Percabangan (If) adalah saat Anda bertemu persimpangan: "JIKA ada macet, belok kanan; JIKA TIDAK, jalan lurus". Perulangan (Loop) adalah saat Anda melewati bundaran: "Putar terus SELAMA Anda belum menemukan pintu keluar yang benar".

1. Percabangan (Conditionals)

Digunakan untuk membuat keputusan berdasarkan kondisi tertentu.

If - Else If - Else

let nilai = 85;

if (nilai >= 90) {
  console.log("Grade: A");
} else if (nilai >= 75) {
  console.log("Grade: B");
} else {
  console.log("Grade: C");
}

2. Switch Statement

Digunakan untuk memeriksa nilai tunggal terhadap banyak kemungkinan "case".

let hari = "Senin";

switch (hari) {
  case "Senin":
    console.log("Hari ini upacara");
    break;
  case "Minggu":
    console.log("Hari ini libur");
    break;
  default:
    console.log("Hari kerja biasa");
}

3. Perulangan (Loops)

Digunakan untuk menjalankan perintah yang sama berulang-ulang tanpa harus menulis ulang kodenya.

For Loop (Perulangan Pasti)

for (let i = 1; i <= 5; i++) {
  console.log("Lari putaran ke-" + i);
}

While Loop (Perulangan Kondisional)

Berjalan terus selama kondisi masih bernilai true.

let i = 1;
while (i <= 3) {
  console.log("Cek kondisi...");
  i++;
}

Do-While Loop

Blok kode akan dijalankan minimal satu kali, baru kemudian dicek kondisinya.

let j = 10;
do {
  console.log("Saya akan muncul sekali walaupun kondisi salah!");
} while (j < 5);

Break & Continue

  • Break: Menghentikan seluruh perulangan secara paksa.
  • Continue: Melewati satu putaran perulangan dan lanjut ke putaran berikutnya.
for (let i = 1; i <= 5; i++) {
  if (i === 3) continue; // Melewati angka 3
  if (i === 5) break;    // Berhenti saat sampai 5
  console.log("Angka: " + i);
}

Kesimpulan

Control flow adalah tulang punggung dari logika program. Tanpanya, aplikasi kita tidak akan bisa membedakan kondisi benar-salah atau melakukan tugas otomatis secara berulang.
