Object

Apa Itu Object?

Object adalah tipe data di JavaScript yang digunakan untuk menyimpan kumpulan data dalam bentuk pasangan key (nama properti) dan value (nilai). Jika Array ibarat daftar belanja, maka Object ibarat identitas lengkap seseorang.

Bayangkan sebuah Buku Kontak di HP Anda. Satu kontak adalah Object. Di dalamnya ada label (Key) seperti "Nama", "Telepon", dan "Email". Nilainya (Value) bisa berupa teks "Budi" atau angka "08123...". Anda mencari informasi bukan berdasarkan nomor urut (index), melainkan berdasarkan labelnya.

Membuat Object di JavaScript

Cara paling umum dan direkomendasikan adalah menggunakan kurung kurawal {} (Object Literal).

// Object Literal
let mahasiswa = {
  nama: "Budi",
  umur: 20,
  jurusan: "Informatika"
};

// Menggunakan new Object()
let siswa = new Object();
siswa.nama = "Siti";
siswa.umur = 19;

Mengakses Property

Ada dua cara utama untuk mengambil data dari sebuah object:

1. Dot Notation (.)

console.log(mahasiswa.nama); // Budi

2. Bracket Notation ([])

Gunakan ini jika nama key mengandung spasi atau diambil dari variabel.

console.log(mahasiswa["jurusan"]); // Informatika

Manipulasi Object

Kita bisa menambah, mengubah, atau menghapus properti kapan saja.

// Menambah
mahasiswa.alamat = "Jakarta";

// Mengubah
mahasiswa.umur = 21;

// Menghapus
delete mahasiswa.jurusan;

Object Bersarang (Nested)

Sebuah object bisa menyimpan object lain di dalamnya.

let user = {
  id: 1,
  profil: {
    hobi: "Coding",
    kota: "Bandung"
  }
};
console.log(user.profil.kota); // Bandung

Looping Properti (for...in)

Untuk melihat semua isi dari sebuah object, kita bisa menggunakan perulangan for...in.

for (let key in mahasiswa) {
  console.log(key + ": " + mahasiswa[key]);
}

Array vs Object

Fitur Array Object
Akses Data Index Angka (0, 1, 2...) Key/Nama Properti
Tanda Kurung Kurung Siku [ ] Kurung Kurawal { }
Tujuan Daftar item berurutan Detail entitas tunggal

Kesimpulan

Object adalah pilar utama dalam JavaScript modern. Hampir semua data yang kita ambil dari API atau kita kelola di Framework (seperti React) berbentuk Object. Memahami cara memanipulasinya akan memudahkan Anda membangun aplikasi yang kompleks.
