Template Literals

Apa Itu Template Literals?

Template Literals adalah fitur modern JavaScript (ES6) yang mempermudah kita bekerja dengan string. Berbeda dengan string tradisional yang menggunakan kutip satu (') atau dua ("), template literals menggunakan karakter backtick ( ` ).

Bayangkan Anda membuat sebuah Surat Undangan. Jika menggunakan cara lama, Anda harus memotong teks satu per satu dan menempelkan nama tamu menggunakan lem (operator +). Dengan Template Literals, Anda memiliki cetakan surat yang sudah ada lubang-lubangnya (placeholder ${}); Anda cukup menyelipkan kartu nama ke lubang tersebut, dan surat pun langsung lengkap.

1. Interpolasi Variabel

Dulu kita harus menggabungkan string dan variabel menggunakan tanda tambah (+). Sekarang, cukup masukkan variabel ke dalam ${...}.

const nama = "Riza";
const hobi = "Coding";

// Cara Lama
console.log("Halo, nama saya " + nama + " dan hobi saya " + hobi);

// Template Literals (Lebih rapi!)
console.log(`Halo, nama saya ${nama} dan hobi saya ${hobi}`);

2. Ekspresi di Dalam String

Kita tidak hanya bisa memasukkan variabel, tapi juga bisa menjalankan logika matematika atau fungsi langsung di dalam string.

const harga = 5000;
const jumlah = 3;

console.log(`Total belanja: Rp${harga * jumlah}`); 
// Output: Total belanja: Rp15000

3. String Multi-Baris (Multiline)

Membuat teks dengan baris baru menjadi sangat mudah. Tidak perlu lagi menggunakan karakter \n di akhir baris.

const pesan = `Halo teman-teman,
Selamat belajar JavaScript ES6.
Semoga artikel ini bermanfaat!`;

console.log(pesan);

4. Membuat Fragmen HTML

Sangat berguna saat Anda ingin menyisipkan elemen HTML secara dinamis melalui JavaScript.

const user = { nama: "Andi", role: "Admin" };

const card = `
  <div class="user-card">
    <h2>${user.nama}</h2>
    <p>Status: ${user.role}</p>
  </div>
`;

Kesimpulan

Template Literals membuat penulisan string jauh lebih bersih dan mengurangi risiko kesalahan pengetikan saat menggabungkan banyak variabel. Ini adalah standar penulisan string yang paling banyak digunakan dalam pengembangan React dan aplikasi modern lainnya.
