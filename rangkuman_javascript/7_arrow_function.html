Arrow Function

Arrow Function diperkenalkan di ES6 (ECMAScript 2015) sebagai cara penulisan function yang lebih singkat, modern, dan memiliki perilaku this yang berbeda.

ðŸ’¡ Analogi

Bayangkan sebuah Busur dan Panah. Menulis function biasa seperti membangun ketapel kayu besar yang butuh banyak ruang. Sedangkan Arrow Function adalah busur modern yang ringan; cukup tarik satu garis lurus (panah =>) dan Anda langsung bisa menembak ke target (hasil/return). Lebih efisien, lebih cepat, dan hemat tempat.

Apa Itu Arrow Function?

Arrow Function adalah alternatif penulisan fungsi yang lebih ringkas. Kita tidak lagi menggunakan kata kunci function, melainkan simbol "panah gemuk" (fat arrow) yaitu =>.

Perbandingan Penulisan

1. Function Biasa

function sapa(nama) {
  return "Halo " + nama;
}

2. Arrow Function

const sapa = (nama) => {
  return `Halo ${nama}`;
};

Variasi Penulisan

A. Tanpa Parameter (Wajib pakai tanda kurung kosong):

const halo = () => console.log("Halo!");

B. Satu Parameter (Boleh hapus tanda kurung):

const kuadrat = x => x * x;

C. Implicit Return (Hanya untuk isi satu baris, otomatis mengembalikan nilai tanpa kata return):

const tambah = (a, b) => a + b; 

console.log(tambah(5, 10)); // 15

Mengembalikan Object

Untuk mengembalikan sebuah object secara langsung (implicit return), Anda harus membungkus object-nya dengan tanda kurung () agar tidak dikira sebagai blok kode {}.

const buatUser = (nama, role) => ({ 
  username: nama, 
  level: role 
});

console.log(buatUser("Riza", "Admin"));

Perbedaan Utama

Fitur Function Biasa Arrow Function
Sintaks Panjang & deskriptif Singkat & minimalis
Hoisting Bisa dipanggil sebelum dibuat Tidak (Harus dibuat dulu)
this Dinamis (tergantung pemanggil) Lexical (tetap pada scope luar)

Kesimpulan

Arrow function adalah pilihan utama di pengembangan JavaScript modern (seperti React) karena sintaksnya yang bersih. Namun, gunakan function biasa jika Anda memerlukan fitur hoisting atau perilaku this yang dinamis.

Di materi berikutnya, kita akan menjelajahi Array: Cara mengelola kumpulan data dengan metode-metode sakti JavaScript!
