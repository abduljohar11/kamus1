Default Parameters

Apa Itu Default Parameters?

Default Parameters adalah fitur ES6 yang memungkinkan kita memberikan nilai cadangan (default) pada parameter fungsi. Jika saat fungsi dipanggil kita tidak mengirimkan nilai (argumen), maka nilai cadangan inilah yang akan digunakan.

Bayangkan Anda memesan Nasi Goreng di warung. Jika Anda bilang "Pak, beli nasi goreng satu", maka penjual akan memberikan level pedas Sedang secara otomatis (ini adalah nilai default). Namun, jika Anda bilang "Pak, beli nasi goreng satu, level Sangat Pedas", maka penjual akan mengikuti instruksi Anda dan mengabaikan nilai default tadi.

Contoh Tanpa Default Parameters

Sebelum adanya ES6, kita harus mengecek secara manual di dalam blok kode fungsi menggunakan pengkondisian if atau operator OR.

function sapa(nama) {
  let user = nama || "Tamu";
  console.log("Halo, " + user);
}

sapa(); // Halo, Tamu
sapa("Riza"); // Halo, Riza

Contoh Dengan Default Parameters

Sekarang, kita bisa langsung menetapkan nilai default tepat di samping nama parameter saat mendefinisikan fungsi.

function sapa(nama = "Tamu") {
  console.log(`Halo, ${nama}`);
}

sapa();      // Output: Halo, Tamu
sapa("Budi"); // Output: Halo, Budi

Cara Kerja Default Parameters

  • Gunakan Default: Jika argumen tidak diisi atau bernilai undefined.
  • Gunakan Argumen: Jika Anda mengirimkan nilai apa pun (String, Number, dll) saat pemanggilan fungsi.

Default Parameters dengan Expression

Hebatnya, nilai default tidak harus berupa teks statis. Kita bisa menggunakan ekspresi matematika atau bahkan variabel lain sebagai default.

function hitungLuas(panjang, lebar = panjang) {
  // Jika lebar tidak diisi, otomatis dianggap persegi (lebar = panjang)
  return panjang * lebar;
}

console.log(hitungLuas(10));     // 100 (10 * 10)
console.log(hitungLuas(10, 5));  // 50 (10 * 5)

Kesimpulan

Default Parameters membuat fungsi kita lebih tangguh (robust) karena dapat mencegah error ketika ada argumen yang lupa dikirimkan. Ini juga membuat kode jauh lebih bersih karena kita tidak perlu lagi menulis banyak pengecekan manual di dalam tubuh fungsi.
