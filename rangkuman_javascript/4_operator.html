Operator

Operator sangat penting karena digunakan untuk melakukan berbagai operasi pada data, seperti perhitungan matematika, perbandingan nilai, hingga logika dalam alur program.

Bayangkan sebuah Kalkulator. Angka yang Anda masukkan adalah Data (Operand), sedangkan tombol plus (+), minus (-), atau sama dengan (=) adalah Operator. Tanpa menekan tombol operator, angka-angka tersebut tidak akan pernah berubah menjadi hasil yang Anda inginkan.

Apa Itu Operator?

Operator adalah simbol khusus yang menginstruksikan komputer untuk melakukan manipulasi tertentu pada satu atau lebih nilai. Nilai yang diolah disebut sebagai operand.

1. Operator Aritmatika

Digunakan untuk perhitungan matematika dasar.

SimbolOperasiContoh
+Penjumlahan10 + 5 = 15
-Pengurangan10 - 5 = 5
*Perkalian10 * 5 = 50
/Pembagian10 / 2 = 5
%Sisa Bagi (Modulo)10 % 3 = 1
**Perpangkatan2 ** 3 = 8

2. Operator Perbandingan

Membandingkan dua nilai dan menghasilkan Boolean (true atau false).

console.log(10 == "10");  // true (hanya cek nilai)
console.log(10 === "10"); // false (cek nilai & tipe data - SANGAT DISARANKAN)
console.log(10 > 5);      // true
console.log(10 != 5);     // true

3. Operator Logika

Digunakan untuk menggabungkan kondisi-kondisi Boolean.

  • && (AND): Benar jika semua kondisi benar.
  • || (OR): Benar jika salah satu kondisi benar.
  • ! (NOT): Membalikkan nilai (true jadi false).
let punyaTiket = true;
let punyaUang = false;

console.log(punyaTiket && punyaUang); // false
console.log(punyaTiket || punyaUang); // true

4. Operator Penugasan (Assignment)

Digunakan untuk memasukkan nilai ke variabel secara ringkas.

let x = 10;
x += 5; // Sama dengan x = x + 5
console.log(x); // 15

5. Operator Ternary

Cara singkat menulis if-else dalam satu baris.

let umur = 20;
let status = (umur >= 18) ? "Dewasa" : "Anak-anak";
console.log(status); // Dewasa

Kesimpulan

Operator adalah alat pertukangan dalam JavaScript. Dengan menguasai operator aritmatika, perbandingan, dan logika, Anda sudah bisa mulai membangun mesin logika dasar dalam program Anda.
