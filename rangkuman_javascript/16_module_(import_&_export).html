Module (import & export)

Apa Itu Module di JavaScript?

Module adalah file JavaScript terpisah yang dapat mengekspor function, object, atau variable agar bisa digunakan di file lain. Fitur ini sangat krusial dalam pengembangan web modern agar kode kita tetap bersih dan terorganisir.

ðŸ’¡ Analogi

Bayangkan Anda sedang bermain Lego. Anda tidak memiliki satu balok besar berbentuk kastil, melainkan banyak kepingan kecil (Module). Kepingan pintu bisa Anda pakai di kastil A, lalu Anda bawa (Export) dan pasang (Import) di kastil B. Dengan begitu, Anda tidak perlu membangun pintu dari nol setiap kali membuat kastil baru.

Cara Export di JavaScript

Agar sebuah fungsi atau variabel bisa dipakai di luar file aslinya, kita harus memberikan izin menggunakan kata kunci export.

1. Named Export

Digunakan jika kita ingin mengekspor banyak hal dari satu file.

// file: math.js
export const PHI = 3.14;

export function tambah(a, b) {
  return a + b;
}

export function kurang(a, b) {
  return a - b;
}

2. Default Export

Digunakan jika file tersebut hanya memiliki satu "nilai utama" yang diekspor. Satu file hanya boleh punya satu default export.

// file: greet.js
export default function greet(nama) {
  return `Halo, ${nama}!`;
}

Cara Import di JavaScript

Gunakan kata kunci import untuk mengambil apa yang sudah diekspor oleh file lain.

Import dari Named Export

Wajib menggunakan kurung kurawal { } dan nama variabel harus persis.

// file: app.js
import { tambah, kurang, PHI } from './math.js';

console.log(tambah(5, 3)); // 8
console.log(PHI);        // 3.14

Import dari Default Export

Tidak perlu menggunakan kurung kurawal dan Anda bebas memberi nama saat mengimpornya.

// file: main.js
import salam from './greet.js'; // Kita beri nama 'salam' padahal aslinya 'greet'

console.log(salam('Budi')); // Halo, Budi!

Ringkasan Perbedaan

  • Named Export: Menggunakan { } saat import, bisa banyak dalam satu file, nama variabel harus sama.
  • Default Export: Tanpa { } saat import, hanya satu per file, nama bebas saat di-import.

Kesimpulan

Module memungkinkan kita membangun aplikasi besar dengan menyusun potongan-potongan kecil kode. Ini mempermudah tim untuk bekerja secara paralel tanpa takut kode saling bertabrakan.
