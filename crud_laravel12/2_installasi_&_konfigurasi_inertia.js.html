Konfigurasi Inertia Server Side

Agar Laravel bisa berkomunikasi dengan React.js secara mulus, kita perlu melakukan konfigurasi dari sisi server terlebih dahulu.

Langkah 1 - Install Inertia Adapter

Jalankan perintah berikut di terminal proyek Laravel kamu untuk menginstal Inertia Server-side Adapter:

composer require inertiajs/inertia-laravel

Langkah 2 - Middleware Konfigurasi

Publish Middleware Inertia agar kita bisa mengontrol data yang dibagikan (shared data) ke frontend:

php artisan inertia:middleware

Daftarkan middleware tersebut di file bootstrap/app.php agar aktif secara global:

->withMiddleware(function (Middleware $middleware): void {
    $middleware->web(append: [
        \App\Http\Middleware\HandleInertiaRequests::class,
    ]);
})

Langkah 3 - Shared Data (Flash Message)

Buka file app/Http/Middleware/HandleInertiaRequests.php. Pada method share, tambahkan logic untuk menangkap pesan notifikasi (flash session):

public function share(Request $request): array
{
    return array_merge(parent::share($request), [
        'flash' => [
            'message' => fn () => $request->session()->get('message'),
        ],
    ]);
}

Langkah 4 - Root View Template

Buat file baru bernama app.blade.php di folder resources/views. File ini akan menjadi wadah utama aplikasi SPA kita:

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0" />
    @viteReactRefresh
    @vite('resources/js/app.jsx')
    @inertiaHead
  </head>
  <body>
    @inertia
  </body>
</html>

Kesimpulan

Proses konfigurasi Server-side telah selesai. Sekarang Laravel sudah siap mengirimkan data ke komponen React. Di tahap berikutnya, kita akan melakukan instalasi Client-side (React & Vite).
