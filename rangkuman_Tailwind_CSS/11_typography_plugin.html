Typography Plugin

Plugin ini mempermudah kita dalam menata konten teks panjang seperti artikel, blog post, dan dokumentasi agar tampilannya langsung indah tanpa banyak menulis utility class secara manual.

Apa Itu Typography Plugin?

Typography Plugin memberikan utility bernama prose. Saat class ini diberikan pada sebuah container, semua elemen HTML standar di dalamnya akan otomatis memiliki styling yang rapi (tipografi, spasi, warna, dan ukuran).

1. Cara Mengaktifkan (v4)

Di Tailwind CSS v4, aktivasi plugin dilakukan langsung di file CSS utama menggunakan directive baru:

@import "tailwindcss";
@plugin "@tailwindcss/typography";

*Catatan: Ini membutuhkan setup CLI, tidak berfungsi via CDN standar.

2. Class Prose & Variasi

Kamu bisa mengatur lebar artikel dan warna elemen menggunakan modifier:

ClassKegunaan
proseMengaktifkan gaya tipografi standar
prose-sm / lg / xlMengatur skala ukuran teks keseluruhan
prose-slate / blueMengubah tema warna elemen (link, heading)
prose-invertMode gelap (untuk background gelap)

3. Elemen yang Terdampak

Setelah membungkus konten dengan class prose, elemen berikut akan otomatis tertata:

  • Judul (<h1> sampai <h6>)
  • Paragraf dan Link (<p>, <a>)
  • Daftar (<ul>, <ol>, <li>)
  • Kutipan (<blockquote>)
  • Tabel (<table>, <th>, <td>)
  • Blok Kode (<pre>, <code>)

Contoh Kode: Artikel

<article class="prose lg:prose-xl prose-slate mx-auto">
  <h1>Judul Artikel Otomatis Rapi</h1>
  <p>Paragraf ini tidak butuh class text-lg atau leading-relaxed.</p>
  
  <ul>
    <li>List otomatis memiliki bullet</li>
    <li>Jarak antar list sudah diatur</li>
  </ul>

  <blockquote>
    "Kutipan ini otomatis memiliki garis samping dan font miring."
  </blockquote>
</article>

Kesimpulan

Typography Plugin adalah "cheat sheet" untuk artikel. Kita tidak perlu pusing memikirkan margin antar paragraf atau warna heading; cukup panggil prose dan biarkan Tailwind yang bekerja.
