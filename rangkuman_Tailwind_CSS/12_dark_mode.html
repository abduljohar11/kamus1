Dark Mode

Tailwind CSS v4 memudahkan implementasi mode gelap melalui varian dark:. Kamu bisa memilih untuk mengikuti pengaturan sistem pengguna atau memberikan kontrol manual (toggle).

1. Mode Sistem (Media Strategy)

Secara default, Tailwind mendeteksi pengaturan sistem (Windows/macOS/Android). Jika pengguna menggunakan tema gelap di perangkatnya, class dengan prefix dark: akan otomatis aktif.

<!-- bg-white di mode terang, bg-slate-900 di mode gelap -->
<div class="bg-white dark:bg-slate-900">
  <h1 class="text-slate-900 dark:text-white">Halo Dunia!</h1>
</div>

2. Mode Manual (Selector Strategy)

Untuk Tailwind v4, jika ingin menggunakan toggle manual, kamu bisa menentukan strategi di dalam file CSS utama menggunakan @theme:

@import "tailwindcss";

@theme {
  /* Menggunakan class .dark pada tag html untuk aktifkan mode gelap */
  --dark-mode: selector; 
}

4. Tips Dark Mode

  • Gunakan transition-colors agar perpindahan warna tidak kaku saat user menekan toggle.
  • Simpan preferensi tema di localStorage agar saat halaman di-refresh, tema tidak kembali ke awal.
  • Gunakan warna Slate atau Zinc untuk background gelap agar kontras teks tetap nyaman di mata (tidak hitam pekat).

Contoh Skrip Toggle

Gunakan JavaScript sederhana ini untuk membuat tombol switch tema:

// Klik tombol untuk tambah/hapus class 'dark' di tag HTML
document.documentElement.classList.toggle('dark');

// Simpan pilihan ke localStorage
localStorage.setItem('theme', 'dark');

Kesimpulan

Mode gelap bukan lagi sekadar tren, tapi kebutuhan untuk kenyamanan mata pengguna. Dengan Tailwind, kamu cukup memikirkan satu warna alternatif dengan prefix dark: untuk setiap elemen visual.
