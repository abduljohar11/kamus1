Grid Layout System

Berbeda dengan Flexbox yang satu dimensi, Grid adalah sistem layout dua dimensi. Artinya, kita bisa mengatur baris dan kolom secara bersamaan dengan sangat presisi.

1. Membuat Grid Dasar

Gunakan class grid pada parent, diikuti dengan jumlah kolom menggunakan grid-cols-{n}.

1
2
3
<!-- Membuat grid dengan 3 kolom -->
<div class="grid grid-cols-3 gap-4">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>

2. Jarak (Gap)

Sama seperti Flexbox, Grid menggunakan gap untuk mengatur jarak antar elemen.

ClassKeterangan
gap-{n}Jarak horizontal dan vertikal sama
gap-x-{n}Hanya jarak horizontal (kolom)
gap-y-{n}Hanya jarak vertikal (baris)

3. Grid Column Span

Kadang kita ingin satu item memakan ruang lebih dari satu kolom. Gunakan col-span-{n}.

Span 2 Kolom
3
4
Span 2 Kolom
<!-- Item pertama memakan 2 kolom -->
<div class="grid grid-cols-3 gap-4">
  <div class="col-span-2 bg-orange-500">Besar</div>
  <div class="bg-blue-500">Kecil</div>
</div>

4. Contoh Lengkap: Layout Galeri

Implementasi grid untuk layout galeri atau dashboard yang rapi.

<div class="grid grid-cols-2 md:grid-cols-4 gap-4 p-4 bg-white rounded-xl shadow">
  <div class="bg-slate-200 h-24 rounded flex items-center justify-center font-bold">1</div>
  <div class="bg-slate-200 h-24 rounded flex items-center justify-center font-bold">2</div>
  <div class="bg-slate-200 h-24 rounded flex items-center justify-center font-bold">3</div>
  <div class="bg-slate-200 h-24 rounded flex items-center justify-center font-bold">4</div>
</div>

Kesimpulan

Grid sangat cocok untuk layout besar (halaman utama, galeri, form), sedangkan Flexbox lebih cocok untuk layout mikro (navbar, isi card, perataan icon). Gunakan keduanya secara kombinasi untuk hasil terbaik.
