Pengenalan & Instalasi Tailwind CSS

Dengan memahami Tailwind CSS, kita bisa membangun antarmuka yang cantik dan responsif hanya dengan menambahkan class-class langsung pada elemen HTML, tanpa perlu menulis CSS secara manual.

Apa Itu Tailwind CSS?

Tailwind CSS adalah sebuah utility-first CSS framework, menyediakan banyak utility class siap pakai untuk mengatur layout, spacing, warna, dan tipografi.

Kenapa Menggunakan Tailwind?

  • Cepat dan Fleksibel: Cukup tambahkan class di HTML.
  • Tanpa Custom CSS: Mengurangi tumpukan file CSS eksternal.
  • Responsive-ready: Kontrol breakpoint langsung di elemen.
  • Produktivitas: Fokus pada logika tampilan di satu tempat.

Cara Instalasi

1. Menggunakan CDN

Metode tercepat untuk belajar atau prototyping sederhana.

<!DOCTYPE html>
<html>
  <head>
    <script src="https://cdn.tailwindcss.com"></script>
  </head>
  <body>
    <h1 class="text-2xl text-green-600 font-semibold">Hello Tailwind!</h1>
  </body>
</html>

2. Menggunakan Tailwind CLI

Direkomendasikan untuk proyek nyata agar performa lebih optimal.

Langkah 1: Setup Folder 
mkdir proyek-tailwind && cd proyek-tailwind
npm init -y
Langkah 2: Instalasi 
npm install tailwindcss @tailwindcss/cli
Langkah 3: Konfigurasi Input CSS

Buat file src/input.css dan masukkan baris berikut:

@import "tailwindcss";
Langkah 4: Jalankan Build 
npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --watch

Kesimpulan

  • Tailwind memindahkan styling langsung ke dalam atribut class HTML.
  • Gunakan CDN untuk eksperimen cepat.
  • Gunakan CLI untuk proyek profesional agar ukuran file CSS tetap kecil.
