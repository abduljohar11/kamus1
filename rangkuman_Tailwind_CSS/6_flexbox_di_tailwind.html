Flexbox Mastery

Flexbox adalah sistem layout satu dimensi yang sangat kuat. Kita bisa mengatur alur elemen secara horizontal maupun vertikal dengan sangat mudah melalui kombinasi class flex, justify, dan items.

1. Justify Content (Main Axis)

Mengatur posisi elemen di sepanjang sumbu utama (Horizontal jika menggunakan flex-row).

justify-center (Tengah Horizontal)
1
2
justify-between (Menyebar ke Ujung)
Kiri
Kanan
<!-- Contoh penggunaan Justify Content -->
<div class="flex justify-between items-center p-4 bg-white">
  <div>Logo</div>
  <div>Menu</div>
</div>

2. Align Items (Cross Axis)

Mengatur posisi elemen di sepanjang sumbu silang (Vertikal jika menggunakan flex-row).

items-start (Atas)
Atas
items-center (Tengah Vertikal)
Tengah

3. Flex Direction (Arah Alur)

Mengubah arah susunan elemen dari baris menjadi kolom. Sangat penting untuk tampilan responsif mobile.

flex-row (Default)
A
B
flex-col (Kolom/Tumpuk)
A
B

4. Teknik "Dead Center"

Cara tercepat membuat konten berada tepat di tengah-tengah kontainer secara horizontal maupun vertikal.

Tengah Sempurna
<!-- Gabungan Justify dan Items Center -->
<div class="flex justify-center items-center h-48 bg-white border">
  <div class="p-6 bg-blue-600 text-white rounded-lg shadow-xl">
    Centered Content
  </div>
</div>

Kesimpulan & Ringkasan

  • Gunakan flex untuk mengaktifkan mode Flexbox pada parent.
  • Gunakan justify-* untuk sumbu utama (Horizontal di row).
  • Gunakan items-* untuk sumbu silang (Vertikal di row).
  • Gunakan flex-col untuk membuat elemen bertumpuk vertikal.
  • Gunakan gap-{n} untuk memberi jarak otomatis antar item.
