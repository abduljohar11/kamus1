Apa Yang Baru di Laravel 12

Laravel 12 hadir dengan berbagai peningkatan performa dan penyederhanaan sintaks. Fokus utamanya adalah membuat kode PHP kamu semakin pendek, namun tetap mudah dibaca.

1. Laravel Starter Kits Terpadu

Laravel 12 memperkenalkan penggabungan installer untuk Breeze dan Jetstream. Kamu kini bisa memilih stack (Blade, Livewire, atau Inertia) dengan lebih fleksibel langsung saat instalasi proyek baru.

laravel new my-app --stack=inertia --dark

2. Peningkatan Eloquent

Eloquent di Laravel 12 semakin cerdas dalam menangani query database yang kompleks. Salah satu fitur barunya adalah Deferrable Validations yang memungkinkan validasi database dilakukan hanya saat data benar-benar akan disimpan.

3. Native PHP 8.4 Support

Laravel 12 memanfaatkan fitur-fitur terbaru dari PHP 8.4, seperti Property Hooks yang memungkinkan kita melakukan get/set logic langsung pada properti class tanpa harus membuat method manual.

// Contoh Property Hooks di PHP 8.4 / Laravel 12
public string $name {
    set => strtolower($value);
    get => ucfirst($value);
}

4. Vite 6 Integration

Secara default, Laravel 12 menggunakan Vite 6 sebagai module bundler. Ini memberikan kecepatan build yang lebih luar biasa, terutama sangat terasa saat kamu mengembangkan aplikasi menggunakan Inertia.js dan React.

5. New Artisan Commands

Beberapa perintah baru ditambahkan untuk mempermudah debugging dan manajemen database langsung dari terminal, termasuk peningkatan pada perintah php artisan model:show.

Kesimpulan

Laravel 12 terus membuktikan dirinya sebagai framework PHP paling modern. Dengan dukungan penuh terhadap PHP 8.4 dan Vite 6, pengembangan aplikasi Fullstack menjadi jauh lebih efisien.
