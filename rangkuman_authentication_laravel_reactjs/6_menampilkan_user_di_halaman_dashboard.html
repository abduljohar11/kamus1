Menampilkan User di Halaman Dashboard

Langkah 1 - Membuat Controller Dashboard

Silahkan teman-teman masuk ke dalam folder project Laravel-nya, kemudian jalankan perintah berikut ini di dalam terminal/CMD.

php artisan make:controller DashboardController -i

Karena di atas kita menambahkan flag -i, maka secara otomatis controller tersebut akan dijadikan sebagai Single Action Controller.

<?php

namespace App\Http\Controllers;

use Illuminate\Http\Request;

class DashboardController extends Controller
{
    /**
     * Handle the incoming request.
     */
    public function __invoke(Request $request)
    {
        return inertia('Dashboard/Index');
    }
}

Langkah 2 - Membuat Route Dashboard

Sekarang kita tambahkan route untuk halaman dashboard. Buka file routes/web.php dan tambahkan kode berikut:

// route dashboard
Route::get('/dashboard', \App\Http\Controllers\DashboardController::class)->middleware('auth');

Untuk memastikan route berhasil ditambahkan, jalankan perintah:

php artisan route:list

Langkah 3 - Membuat Halaman Dashboard

Silahkan buat folder baru Dashboard di dalam resources/js/Pages, kemudian buat file Index.jsx:

import React from 'react';
import { Head, usePage, router } from '@inertiajs/react';

export default function Dashboard() {

    // destruct props "auth" dari Middleware
    const { auth } = usePage().props;

    const logout = () => {
        router.post('/logout');
    }

    return (
        <>
            <Head title="Dashboard - My App" />
            <div className="flex items-center justify-center min-h-screen">
                <div className="w-full max-w-2xl p-8 bg-white rounded-2xl shadow-xl border border-slate-200">
                    <div className="flex justify-between items-center mb-8">
                        <h1 className="text-2xl font-black uppercase tracking-tight text-slate-800">Dashboard</h1>
                        <button onClick={logout} className="bg-red-500 text-white px-4 py-2 rounded-lg font-bold text-xs hover:bg-red-600 transition">
                            LOGOUT
                        </button>
                    </div>

                    <div className="bg-blue-50 p-6 rounded-xl border border-blue-100">
                        <p className="text-slate-600 mb-2">Selamat Datang,</p>
                        <h2 className="text-3xl font-black text-blue-900">{auth.user.name}</h2>
                        <p className="text-blue-700 font-medium">{auth.user.email}</p>
                    </div>

                    <div className="mt-8 pt-8 border-t border-slate-100 text-center text-slate-500 text-sm">
                        Kamu berhasil login menggunakan Laravel + Inertia.js + React
                    </div>
                </div>
            </div>
        </>
    );
}

Langkah 4 - Uji Coba Halaman Dashboard

Pastikan npm run dev masih berjalan. Sekarang akses:

http://localhost:8000/dashboard

Jika berhasil, kamu akan melihat nama dan email kamu yang diambil langsung dari data autentikasi yang dikirim melalui HandleInertiaRequests middleware.

Sampai disini kita telah berhasil membangun alur autentikasi lengkap mulai dari Register, Login, hingga Dashboard. Selamat!
