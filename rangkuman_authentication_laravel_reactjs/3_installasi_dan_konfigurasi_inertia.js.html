Installasi dan Konfigurasi Inertia.js (Client-Side)

Langkah 1 - Installasi Node.js

Pertama, teman-teman harus memiliki Node.js yang sudah terinstall di dalam komputer, jika belum maka bisa lakukan proses installasi Node.js terlebih dahulu dari situs resminya : https://nodejs.org/en/

Untuk memastikan Node.js sudah terinstall, silahkan jalankan perintah berikut ini di dalam terminal/CMD.

node -v
npm -v

Langkah 2 - Installasi Inertia.js "client-side"

Setelah Node.js berhasil terinstall, maka sekarang kita akan lanjutkan melakukan installasi library-library yang dibutuhkan, termasuk Inertia, React, dan Vite.

Silahkan teman-teman masuk ke dalam folder project-nya, kemudian jalankan perintah berikut ini di dalam terminal/CMD.

npm install @inertiajs/react @radix-ui/react-slot lucide-react clsx tailwind-merge

Dan karena kita menggunakan React, kita juga perlu menambahkan plugin React untuk Vite:

npm install @vitejs/plugin-react --save-dev

Setelah proses installasi di atas selesai, silahkan buka file package.json, pada bagian dependencies pastikan sudah ada library berikut:

"dependencies": {
    "@inertiajs/react": "^1.0.0",
    "react": "^18.2.0",
    "react-dom": "^18.2.0"
}

Langkah 3 - Konfigurasi App.jsx

Sekarang silahkan rename file resources/js/app.js menjadi resources/js/app.jsx, kemudian masukkan kode berikut ini di dalamnya.

import './bootstrap';
import '../css/app.css';

import { createRoot } from 'react-dom/client';
import { createInertiaApp } from '@inertiajs/react';
import { resolvePageComponent } from 'laravel-vite-plugin/inertia-helpers';

const appName = window.document.getElementsByTagName('title')[0]?.innerText || 'Laravel';

createInertiaApp({
    title: (title) => `${title} - ${appName}`,
    resolve: (name) => resolvePageComponent(`./Pages/${name}.jsx`, import.meta.glob('./Pages/**/*.jsx')),
    setup({ el, App, props }) {
        const root = createRoot(el);
        root.render(<App {...props} />);
    },
    progress: {
        color: '#4B5563',
    },
});

Setelah berhasil melakukan inisialisasi, jangan lupa buat folder baru dengan nama Pages di dalam folder resources/js untuk menyimpan komponen React kita nanti.

Langkah 4 - Konfigurasi Vite

Sekarang, silahkan buka file vite.config.js, kemudian ubah kode-nya menjadi seperti berikut ini untuk mendukung React dan Inertia.

import { defineConfig } from 'vite';
import laravel from 'laravel-vite-plugin';
import react from '@vitejs/plugin-react';

export default defineConfig({
    plugins: [
        laravel({
            input: 'resources/js/app.jsx',
            refresh: true,
        }),
        react(),
    ],
});

Langkah 5 - Compile Assets

Terakhir, kita perlu menjalankan Vite agar aset kita di-compile secara real-time saat kita melakukan pengembangan.

npm run dev

Jika berhasil, terminal akan menampilkan informasi bahwa Vite sedang berjalan. Kamu bisa membuka alamat localhost yang disediakan di browser.

INFORMASI PENTING : saat proses development, jangan pernah mematikan proses Vite di atas. Karena Vite akan otomatis melakukan compiling ketika terdapat perubahan pada kode React kamu.
