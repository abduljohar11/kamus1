PHP Pengenalan

Apa Itu PHP?

PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side yang dirancang khusus untuk pengembangan web. Awalnya dibuat oleh Rasmus Lerdorf pada tahun 1994.

Kenapa Harus Belajar PHP?

Banyak platform besar seperti WordPress, Facebook, dan Wikipedia dibangun dengan PHP. Selain itu, PHP juga punya banyak framework keren seperti Laravel yang mempercepat pengembangan.

Langkah 1 - Cara Install PHP

Kita bisa menggunakan paket seperti XAMPP atau Laragon yang sudah menyertakan PHP, Apache, dan MySQL.

Opsi 1: Install via XAMPP (Windows/macOS)

  1. Buka situs resmi: Apache Friends
  2. Download XAMPP sesuai sistem operasi Anda.
  3. Jalankan installer, klik "Next" sampai selesai.
  4. Buka XAMPP Control Panel dan start Apache.
Tips: XAMPP sangat cocok untuk pemula karena instalasinya yang mudah.

Opsi 2: Install Manual (Linux)

sudo apt update
sudo apt install php
php -v

Langkah 2 - Hello World Dengan PHP

Mari kita buat program pertama kita. Ikuti instruksi di bawah ini:

// 1. Buat folder project
mkdir belajar-php
cd belajar-php

// 2. Buat file index.php dan isi dengan:
<?php
    echo "Hello, World!";
?>

// 3. Jalankan built-in server
php -S localhost:8000

Buka browser dan kunjungi http://localhost:8000 untuk melihat hasilnya.

Kesimpulan

Kita sudah berhasil menginstal PHP dan menjalankan program "Hello World". Di seri berikutnya, kita akan membahas Struktur Dasar Program PHP.
