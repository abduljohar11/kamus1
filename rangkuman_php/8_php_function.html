PHP Function (Fungsi)

Apa Itu Fungsi?

Fungsi adalah sekumpulan instruksi yang dibungkus dalam satu nama. Tujuannya adalah agar kode tersebut bisa dipanggil berkali-kali tanpa harus menulis ulang.

Analogi Robot Pelayan:

Bayangkan kamu punya robot. Kamu mengajari robot cara "Membuat Kopi". Sekali robot itu paham, kamu cukup berteriak "Buat Kopi!", dan robot akan menjalankan semua langkahnya secara otomatis.

1. Membuat Fungsi Dasar

Fungsi dibuat menggunakan kata kunci function. Kode di dalamnya tidak akan berjalan sampai kamu memanggil namanya.

<?php
  // Mendefinisikan fungsi
  function sapaGuru() {
      echo "Selamat pagi, Pak/Bu Guru! Semangat mengajar hari ini.<br>";
  }

  // Memanggil fungsi
  sapaGuru(); 
?>

2. Parameter (Input untuk Fungsi)

Parameter adalah variabel "tampungan" untuk menerima data dari luar agar fungsi lebih dinamis.

<?php
  function sapaSiswa($nama, $status) {
      echo "Halo $nama, kamu hari ini dinyatakan $status.<br>";
  }

  sapaSiswa("Andi", "Hadir");
  sapaSiswa("Budi", "Izin");
?>

3. Return Value (Hasil Balik)

Terkadang kita ingin fungsi tidak hanya mencetak teks, tapi memberikan sebuah hasil yang bisa kita simpan ke variabel lain.

[Image of a function machine with input and output] 
<?php
  // Fungsi untuk menghitung nilai akhir
  function hitungNilai($tugas, $uts, $uas) {
      $total = ($tugas + $uts + $uas) / 3;
      return $total; // Mengembalikan hasil perhitungan
  }

  $nilaiBudi = hitungNilai(80, 75, 90);
  echo "Nilai akhir Budi adalah: " . $nilaiBudi;
?>

4. Parameter Default

Kamu bisa memberikan nilai cadangan jika user lupa memasukkan data saat memanggil fungsi.

<?php
  function setKkm($nilai = 75) {
      echo "KKM saat ini adalah: $nilai <br>";
  }

  setKkm();    // Menggunakan default (75)
  setKkm(80);  // Menggunakan nilai baru (80)
?>

Kesimpulan

Fungsi membuat kode kamu DRY (Don't Repeat Yourself). Alih-alih menulis If-Else yang sama di setiap halaman, kamu cukup buat satu fungsi dan panggil berkali-kali. Selamat! Kamu sudah menguasai fondasi logika PHP.
