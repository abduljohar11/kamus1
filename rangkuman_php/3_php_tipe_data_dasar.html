PHP Tipe Data Dasar

Kalau kita mau menyimpan nilai ke dalam variabel, maka kita perlu tahu tipe data apa yang sesuai. Yuk, kita bahas satu per satu!

Apa Itu Tipe Data?

Tipe data adalah jenis nilai yang bisa disimpan dalam sebuah variabel. Di PHP, tipe data akan menentukan bagaimana data tersebut disimpan dan diproses.

<?php
  $nama = "Andi";       // String
  $umur = 25;           // Integer
  $tinggi = 1.75;       // Float
  $isBelajar = true;    // Boolean
?>

Jenis-Jenis Tipe Data Dasar

  • 1. String

    Digunakan untuk menyimpan teks atau kalimat. Ditulis di antara tanda kutip ("" atau '').

    $nama = "Siti";
echo "Halo, nama saya $nama";
  • 2. Integer

    Bilangan bulat, tanpa koma atau titik desimal.

    $umur = 21;
echo $umur;
  • 3. Float (Double)

    Bilangan desimal (mengandung koma).

    $berat = 52.7;
echo $berat;
  • 4. Boolean

    Nilainya hanya dua: true (benar) atau false (salah). Cocok untuk logika kondisi.

    $isOnline = true;
$isLogin = false;
  • 5. Array

    Digunakan untuk menyimpan banyak nilai dalam satu variabel.

    $hobi = ["membaca", "menulis", "ngoding"];
echo $hobi[0]; // Output: membaca
  • 6. Null

    Variabel yang tidak memiliki nilai apa pun (kosong).

    $data = null;

Cek Tipe Data

PHP punya fungsi var_dump() untuk melihat isi dan detail tipe data dari sebuah variabel. Fungsi ini sangat berguna saat proses debugging.

$angka = 100;
var_dump($angka); // Output: int(100)

Kesimpulan

Tipe data adalah pondasi penting dalam pemrograman. Kita sudah mengenal: String, Integer, Float, Boolean, Array, dan Null. Di artikel selanjutnya, kita akan membahas lebih dalam tentang penggunaan Variabel.
