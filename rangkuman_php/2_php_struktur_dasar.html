PHP Struktur Dasar

Cara Menulis Kode PHP

PHP adalah bahasa scripting yang biasanya ditulis di dalam file .php. Untuk menulis kode PHP, kita harus membukanya dengan tag <?php dan menutupnya dengan ?>.

Contohnya:

<?php
  // Ini adalah kode PHP
  echo "Halo, Dunia!";
?>

Semua kode PHP harus ditulis di antara tag tersebut agar dapat diproses oleh server.

Komentar di PHP

Komentar digunakan untuk memberikan catatan di dalam kode supaya lebih mudah dimengerti. Komentar tidak akan dijalankan oleh sistem.

<?php
  // Ini komentar satu baris menggunakan double slash

  # Ini juga komentar satu baris menggunakan tanda pagar

  /*
  Ini komentar
  lebih dari satu baris (multiline)
  */
?>

Statement & Sintaks Dasar

Setiap perintah dalam PHP biasanya diakhiri dengan titik koma ;. Jika lupa, PHP akan memberikan pesan error.

<?php
  echo "Belajar PHP itu seru!";
  echo "Ayo lanjut belajar!";
?>

Struktur Program PHP Sederhana

Umumnya, struktur dasar program PHP terdiri dari tag pembuka, deklarasi variabel atau logika, dan output.

<?php
  $nama = "Budi";
  $umur = 20;

  echo "Halo, nama saya $nama. Saya berumur $umur tahun.";
?>

Penjelasan:

  • $nama dan $umur adalah variabel untuk menyimpan data.
  • echo digunakan untuk menampilkan teks atau nilai variabel ke layar.

Menggabungkan HTML + PHP

PHP sangat fleksibel karena bisa disisipkan langsung di dalam kode HTML.

<!DOCTYPE html>
<html>
<head>
    <title>Belajar PHP</title>
</head>
<body>
    <h1>Selamat Datang!</h1>
    <p>
        <?php echo "Hari ini kita belajar struktur program PHP."; ?>
    </p>
</body>
</html>

Kesimpulan

Kita sudah belajar cara menulis tag PHP, membuat komentar, memahami titik koma, serta menggabungkan PHP dengan HTML. Di artikel berikutnya, kita akan belajar tentang Tipe Data Dasar di PHP.
