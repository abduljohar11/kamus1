PHP Konstanta

Konstanta memiliki konsep yang mirip dengan variabel, namun memiliki perbedaan penting: nilainya tidak dapat diubah setelah ditetapkan. Konstanta sangat cocok untuk menyimpan nilai konfigurasi yang tetap.

Apa Itu Konstanta?

Konstanta adalah sebuah nama untuk menyimpan nilai tetap. Umumnya digunakan untuk menyimpan nilai konfigurasi seperti nama aplikasi, versi, atau batas maksimal login yang tidak boleh berubah selama program berjalan.

Cara Mendefinisikan Konstanta

PHP menyediakan dua cara utama, yaitu menggunakan fungsi define() dan kata kunci const.

1. Menggunakan define()

<?php
  define("APP_NAME", "Belajar PHP");
  define("APP_VERSION", "1.0");

  echo APP_NAME;    // Output: Belajar PHP
  echo APP_VERSION; // Output: 1.0
?>

Konstanta tidak menggunakan tanda dolar ($) dan secara konvensional ditulis dengan HURUF KAPITAL.

2. Menggunakan const

Pendekatan ini sering digunakan di dalam class, namun juga bisa digunakan secara global sejak PHP 5.3.

<?php
  const AUTHOR = "SantriKoding";
  echo AUTHOR; // Output: SantriKoding
?>

Karakteristik Konstanta

  • Nilainya tidak dapat diubah setelah didefinisikan.
  • Tidak menggunakan tanda $ di depan namanya.
  • Bersifat global (dapat diakses dari mana saja dalam skrip).
  • Hanya dapat menyimpan tipe data skalar (string, integer, float, boolean).

Studi Kasus

Misalkan kita mengatur batas percobaan login maksimal di aplikasi:

<?php
  define("MAX_LOGIN_ATTEMPTS", 3);

  $percobaan = 2;

  if ($percobaan >= MAX_LOGIN_ATTEMPTS) {
      echo "Akun dikunci, terlalu banyak percobaan login.";
  } else {
      echo "Masih dapat mencoba login.";
  }
?>

Kesimpulan

Konstanta membuat program lebih stabil dan aman dari perubahan nilai yang tidak disengaja. Di materi selanjutnya, kita akan mempelajari Operator dalam PHP.
