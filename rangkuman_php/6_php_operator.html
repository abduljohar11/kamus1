PHP Operator & Penerapan

Operator adalah simbol khusus untuk melakukan operasi terhadap nilai (operand). Di bawah ini adalah ringkasan jenis operator dan contoh penerapannya dalam kasus nyata.

1. Operator Aritmatika

Digunakan untuk perhitungan matematika dasar:

  • + Penjumlahan
  • - Pengurangan
  • * Perkalian
  • / Pembagian
  • % Modulo (Sisa Bagi)
<?php
  // Contoh Kasus: Kasir Belanja
  $harga_barang = 50000;
  $jumlah_beli = 3;
  $diskon = 10000;

  $total_awal = $harga_barang * $jumlah_beli;
  $total_akhir = $total_awal - $diskon;

  echo "Total Belanja: Rp " . $total_akhir; 
  // Output: Total Belanja: Rp 140000
?>

2. Operator Perbandingan

Menghasilkan Boolean (true/false) untuk mengecek kondisi:

  • == Sama dengan
  • != Tidak sama dengan
  • > Lebih besar
  • < Lebih kecil
  • >= Lebih besar atau sama dengan
<?php
  // Contoh Kasus: Cek Kelulusan (KKM)
  $nilai_siswa = 80;
  $kkm = 75;

  $is_lulus = $nilai_siswa >= $kkm;

  if ($is_lulus) {
      echo "Selamat, Anda Lulus!";
  } else {
      echo "Maaf, silakan remedial.";
  }
?>

3. Operator Logika

Menggabungkan beberapa ekspresi logika sekaligus:

  • && (AND) Benar jika semua kondisi terpenuhi
  • || (OR) Benar jika salah satu kondisi terpenuhi
  • ! (NOT) Membalikkan hasil logika
<?php
  // Contoh Kasus: Validasi Login Multi-Syarat
  $email = "guru@sekolah.com";
  $password = "rahasia123";
  $is_active = true;

  if ($email == "guru@sekolah.com" && $password == "rahasia123" && $is_active) {
      echo "Login Berhasil!";
  } else {
      echo "Akun tidak ditemukan atau tidak aktif.";
  }
?>

4. Operator Penugasan

Digunakan untuk memberi nilai ke variabel secara efisien:

  • = Sama dengan (Isi nilai)
  • += Tambah dan tugaskan
  • .= Gabung teks (String Concatenation)
<?php
  // Contoh Kasus: Akumulasi Skor
  $skor = 0;
  $skor += 100; // Tambah 100 ke skor saat ini
  $skor += 50;  // Skor sekarang 150

  $pesan = "Skor Anda adalah ";
  $pesan .= $skor; // Gabung teks dan variabel skor

  echo $pesan; // Output: Skor Anda adalah 150
?>

Kesimpulan

Operator adalah alat utama untuk mengolah data dan membangun logika program. Penggabungan operator aritmatika, perbandingan, dan logika akan membentuk "otak" dari aplikasi yang kamu bangun. Selanjutnya, kita akan membahas Control Flow untuk mengarahkan alur program.
