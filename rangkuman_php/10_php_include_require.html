PHP Include & Require

Bayangkan kamu sedang membangun rumah dari Lego. Daripada membuat seluruh rumah dalam satu cetakan besar, kamu membaginya menjadi bagian-bagian kecil: atap, pintu, dan dinding. Di PHP, include dan require adalah cara kita "memasang" bagian-bagian tersebut ke file utama.

1. Include (Opsional)

Gunakan ini untuk bagian yang jika "hilang" tidak akan merusak seluruh aplikasi, seperti Header atau Footer.

  • Jika file tidak ada: Muncul Warning.
  • Hasilnya: Website tetap jalan meskipun bagian tersebut kosong.
<?php
  include 'header.php'; 
  echo "Selamat datang di Dashboard Guru!";
  include 'footer.php';
?>

2. Require (Wajib)

Gunakan ini untuk file yang sangat penting, seperti file Koneksi Database atau Kumpulan Fungsi Nilai.

  • Jika file tidak ada: Muncul Fatal Error.
  • Hasilnya: Website langsung berhenti (mati) demi keamanan data.
<?php
  require 'config/database.php'; // WAJIB ADA!
  require 'functions/penilaian.php';
?>

3. Versi _once (Sekali Saja)

Terkadang kita tidak sengaja memanggil file yang sama berkali-kali. Agar tidak terjadi error "Redeclare" (mendefinisikan ulang fungsi), gunakan akhiran _once.

<?php
  require_once 'fungsi_guru.php';
  require_once 'fungsi_guru.php'; // PHP akan mengabaikan panggilan kedua ini
?>

Kesimpulan

Memisah file membuat kodinganmu rapi, mudah dicari, dan profesional. Gunakan require_once untuk file logika (fungsi/db) dan include untuk potongan desain HTML.
