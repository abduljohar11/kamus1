PHP Cookie

Apa Itu Cookie?

Cookie adalah data kecil yang dititipkan oleh server untuk disimpan di dalam Browser Pengguna. Jika Session hilang saat browser ditutup, Cookie bisa bertahan berhari-hari, berbulan-bulan, bahkan bertahun-tahun.

Analogi Kartu Member:
Session itu seperti gelang wahana (langsung dibuang), sedangkan Cookie itu seperti Kartu Member Alfamart. Kartunya kamu bawa pulang di dompet (Browser), dan besok saat kamu datang lagi, kasir (Server) bisa tahu siapa kamu hanya dengan melihat kartu itu.

1. Membuat Cookie

Gunakan fungsi setcookie(). Ingat: fungsi ini harus diletakkan sebelum tag HTML apapun.

<?php
  // Nama Cookie, Nilai, Waktu Kadaluarsa (Sekarang + 1 Hari)
  $waktu = time() + (60 * 60 * 24); 
  setcookie("user_pref", "dark_mode", $waktu);

  echo "Preferensi tampilan disimpan!";
?>

2. Mengakses & Menghapus

Akses menggunakan variabel superglobal $_COOKIE. Untuk menghapus, cukup setel waktunya ke masa lalu.

<?php
  // Membaca Cookie
  if(isset($_COOKIE['user_pref'])) {
      echo "Mode Anda: " . $_COOKIE['user_pref'];
  }

  // Menghapus Cookie (Set waktu ke 1 jam yang lalu)
  setcookie("user_pref", "", time() - 3600);
?>

Cookie vs Session

Fitur Session Cookie
Penyimpanan Server (Aman) Browser (Kurang Aman)
Masa Aktif Habis saat browser tutup Bisa diatur manual (Lama)
Kegunaan Login, Keranjang Belanja Ingat Saya, Tema Web

Kesimpulan

Gunakan Session untuk data rahasia dan sementara. Gunakan Cookie untuk data yang tidak rahasia dan perlu diingat dalam jangka waktu lama. Selamat! Kamu sudah menyelesaikan dasar-dasar PHP!
