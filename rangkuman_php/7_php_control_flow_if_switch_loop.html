Detail Logika PHP

1. Memahami Struktur Percabangan

Percabangan digunakan agar program bisa memilih jalan mana yang harus ditempuh berdasarkan kondisi data.

  • if: Pintu gerbang pertama. Digunakan untuk memeriksa satu kondisi utama. Jika kondisi ini benar (true), kodenya dijalankan.
  • elseif: Cabang alternatif. Digunakan jika kondisi if pertama salah, tapi kamu masih punya kondisi lain untuk diperiksa sebelum menyerah.
  • else: Jalan terakhir. Digunakan untuk menangkap semua kemungkinan yang tidak memenuhi syarat if maupun elseif.
<?php
  $absensi = 70; // Persentase kehadiran siswa

  if ($absensi >= 80) {
      // Dijalankan jika absensi 80 ke atas
      echo "Status: Diizinkan ikut ujian.";
  } elseif ($absensi >= 60) {
      // Dijalankan jika absensi di bawah 80 tapi masih di atas/sama dengan 60
      echo "Status: Butuh tugas tambahan sebelum ujian.";
  } else {
      // Dijalankan jika semua kondisi di atas salah (di bawah 60)
      echo "Status: Tidak diizinkan ikut ujian.";
  }
?>

2. Menguasai Switch Case

Gunakan switch sebagai alternatif yang lebih rapi dari if...elseif jika kamu ingin membandingkan satu variabel tunggal dengan banyak nilai tetap yang spesifik.

Kenapa pakai Switch?

  • Lebih Cepat Dibaca: Struktur kodenya vertikal dan bersih.
  • Efisien: PHP hanya mengevaluasi variabel satu kali di awal.
  • Ideal untuk Menu: Sangat cocok untuk sistem role, navigasi menu, atau pilihan kategori.
<?php
  $role = "guru";

  switch ($role) {
      case "admin":
          echo "Hak akses: Kelola semua data sekolah.";
          break; // Menghentikan pengecekan jika cocok

      case "guru":
          echo "Hak akses: Input nilai dan absen siswa.";
          break;

      case "siswa":
          echo "Hak akses: Lihat nilai pribadi.";
          break;

      default:
          // Dijalankan jika tidak ada case yang cocok
          echo "Akses ditolak: Role tidak dikenal.";
          break;
  }
?>

Penjelasan Komponen:

  • switch ($variabel)
    Ini adalah bagian "pemeriksaan utama". PHP akan mengambil nilai dari variabel ini dan membawanya ke bawah untuk dibandingkan dengan setiap case.
  • case "nilai":
    Ini adalah target perbandingannya. Jika isi variabel sama dengan nilai di sini, maka kode di bawahnya akan dijalankan.
  • break;
    Wajib Ada! Tanpa break, PHP akan terus menjalankan case di bawahnya meskipun nilainya tidak cocok (fenomena ini disebut fall-through).
  • default:
    Ini adalah "jaring pengaman" terakhir. Fungsinya sama seperti else, yaitu menangani kondisi jika tidak ada satu pun target yang cocok.

3. Perulangan (Looping) Otomatis

Perulangan adalah perintah untuk melakukan tugas yang sama berkali-kali. Bayangkan perulangan sebagai "Otomatisasi Tugas" agar kamu tidak perlu menulis kode yang sama secara manual.

A. For Loop (Si Mesin Fotokopi)

Analogi: Seperti menyetel mesin fotokopi untuk mencetak 10 lembar. Kamu sudah tahu angka pastinya sebelum menekan tombol START.

Rumus Baku For Loop:

for ($awal; $syarat; $langkah) { // Perintah }
  • Awal: Titik mulai (Contoh: $i = 1)
  • Syarat: Sampai kapan? (Contoh: $i <= 10)
  • Langkah: Cara jalan (Contoh: $i++ artinya tambah satu tiap putaran)
<?php
  // Perintah: Buat 10 nomor urut absen
  for ($i = 1; $i <= 10; $i++) {
      echo "Nomor Absen: $i <br>";
  }
?>

B. While Loop (Si Penjaga Gerbang)

Analogi: Seperti penjaga gerbang sekolah yang hanya membolehkan masuk selama siswa memakai seragam. Ia tidak tahu berapa siswa yang datang, pokoknya selama syarat terpenuhi, pintu dibuka.

Rumus Baku While Loop:

while ($syarat_masih_benar) { // Perintah; $perubahan; } 
<?php
  $stok_buku = 5;

  while ($stok_buku > 0) {
      echo "Buku dibagikan. Sisa stok: $stok_buku <br>";
      $stok_buku--; // Wajib ada agar tidak terjadi perulangan selamanya
  }
?>

C. Foreach Loop (Si Pembagi Tugas)

Analogi: Seperti guru yang membagikan hasil ujian ke tumpukan kertas di meja. Guru tidak perlu menghitung, cukup ambil satu-satu sampai tumpukannya habis.

Rumus Baku Foreach:

foreach ($tumpukan_data as $data_satuan) { // Perintah } 
<?php
  $daftar_siswa = ["Budi", "Siti", "Andi"];

  // Membongkar array satu per satu
  foreach ($daftar_siswa as $nama) {
      echo "Nama Siswa: $nama <br>";
  }
?>

Kesimpulan Praktis

Gunakan If-Else untuk logika yang kompleks (seperti membandingkan angka), gunakan Switch untuk pilihan menu yang rapi, dan gunakan Foreach jika kamu berurusan dengan daftar data (array).

Di materi selanjutnya, kita akan belajar Function untuk membungkus logika ini agar bisa dipanggil berkali-kali.
