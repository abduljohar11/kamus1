PHP Variabel

Apa Itu Variabel?

Variabel adalah sebuah wadah untuk menyimpan nilai. Nilai ini bisa berupa teks, angka, array, hingga boolean. Di PHP, variabel bersifat dinamis, artinya kita tidak perlu mendeklarasikan tipe datanya secara eksplisit.

Semua variabel di PHP wajib diawali dengan tanda dolar ($).

<?php
  $nama = "Eka";
  $umur = 22;

  echo "Halo, nama saya $nama dan saya berumur $umur tahun.";
?>

Aturan Penamaan Variabel

PHP memiliki aturan baku dalam pemberian nama variabel:

  • Wajib diawali dengan tanda $.
  • Hanya boleh mengandung huruf, angka, dan underscore (_).
  • Tidak boleh diawali dengan angka.
  • Bersifat Case-Sensitive ($hobi berbeda dengan $Hobi).

✅ Valid:

$nama, $_umur, $skor_1

❌ Tidak Valid:

$1nama, $nama-depan, $biaya!

Menampilkan Nilai Variabel

Ada dua cara umum untuk menampilkan variabel: langsung di dalam tanda kutip dua atau menggunakan operator titik (.) untuk penggabungan (konkatenasi).

<?php
  $kota = "Yogyakarta";

  // Cara 1: Langsung dalam kutip dua
  echo "Saya tinggal di $kota";

  // Cara 2: Konkatenasi (Operator Titik)
  echo "Saya tinggal di " . $kota;
?>

Variabel Dinamis

PHP mendukung variable variables, di mana nama sebuah variabel bisa ditentukan oleh isi variabel lain.

<?php
  $nama = "user";
  $user = "Dina";

  echo $$nama; // Output: Dina
?>

Simbol $$nama berarti kita memanggil variabel yang namanya adalah isi dari $nama (yaitu $user).

Kesimpulan

Variabel adalah pondasi dalam mengelola data di PHP. Pastikan selalu mengikuti aturan penamaan agar kode tidak error. Di materi selanjutnya, kita akan membahas tentang Konstanta.
