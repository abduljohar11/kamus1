PHP Array & Laravel

Di dunia Laravel, Array adalah cara utama kita mengemas data dari Database untuk dikirim ke React. Memahami struktur Array berarti kamu memahami bagaimana data mengalir di aplikasi modern.

1. Associative Array (Data Satuan)

Bayangkan satu baris data siswa di database. Isinya bukan cuma nama, tapi ada ID, Kelas, dan Alamat. Kita menggunakan Key (Label) agar data tidak tertukar.

Analogi: Seperti Label Nama pada laci. Kamu tidak mencari barang berdasarkan "laci nomor 1", tapi berdasarkan tulisan "Laci Kaus Kaki". 
<?php
  // Representasi satu data user/guru di Laravel
  $guru = [
      "id" => 1,
      "nama" => "Pak Ismail",
      "mapel" => "Informatika"
  ];

  echo $guru["nama"]; // Output: Pak Ismail
?>

2. Multidimensional Array (Data Massal)

Inilah bentuk data yang akan kamu kirim dari Laravel Controller ke React. Isinya adalah daftar dari banyak baris data.

INFO LARAVEL:

Hasil dari Siswa::all() di Laravel akan menghasilkan struktur seperti ini (dalam bentuk Collection/Array).

<?php
  $siswa_list = [
      ["id" => 1, "nama" => "Andi", "nilai" => 80],
      ["id" => 2, "nama" => "Siti", "nilai" => 95],
      ["id" => 3, "nama" => "Budi", "nilai" => 70]
  ];

  // Mengambil nama siswa kedua (Siti)
  echo $siswa_list[1]["nama"]; 
?>

3. Array dalam Laravel Inertia

Nantinya, di Laravel Controller, kamu akan mengirimkan array ini ke React menggunakan fungsi Inertia::render.

<?php
  // Gambaran kodingan kamu di Laravel nanti
  return Inertia::render('Siswa/Index', [
      'siswa' => $siswa_list,
      'title' => 'Daftar Nilai Siswa'
  ]);
?>

4. Tips Navigasi Array

  • foreach : Digunakan di PHP untuk memproses data sebelum dikirim.
  • json_encode() : Fungsi PHP yang mengubah Array kamu menjadi format yang dimengerti oleh JavaScript/React.

Kesimpulan

Kuasai cara mengambil data dari Multidimensional Array. Jika kamu sudah lancar memanggil $data[0]['nama'], maka belajar Laravel akan terasa jauh lebih mudah.
