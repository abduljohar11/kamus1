Install & Setup Environment Laravel

Panduan langkah demi langkah inisialisasi project Laravel baru menggunakan Composer tradisional maupun Laravel Sail berbasis Docker.

Butuh Referensi Lebih Detail?

Buka rujukan instalasi, konfigurasi server, dan prasyarat sistem langsung dari core developer.

Dokumentasi Resmi Laravel

Metode 1: Via Composer (Tradisional)

Cocok jika local machine sudah terinstall PHP dan MySQL secara native.

1 Prasyarat Sistem

Pastikan komputer Anda sudah terinstall PHP >= 8.2 dan Composer terbaru.

2 Jalankan Perintah Create-Project

Buka terminal/CLI, arahkan ke direktori web server Anda, lalu jalankan perintah berikut:

composer create-project laravel/laravel laravel-api

3 Nyalakan Server Local

Masuk ke folder hasil install, lalu jalankan server bawaan Artisan:

cd laravel-api
php artisan serve

Aplikasi kini bisa diakses melalui URL http://127.0.0.1:8000.

Metode 2: Via Laravel Sail (Docker)

Solusi isolasi environment modern tanpa perlu menginstall PHP/MySQL secara lokal di OS utama.

1 Prasyarat Sistem

Pastikan runtime Docker Desktop sudah terpasang dan sedang berjalan di latar belakang.

2 Inisialisasi Menggunakan curl / bash

Jalankan perintah ini di terminal untuk mengunduh container Laravel Sail dengan service database mysql:

Linux / macOS / WSL2: 
curl -s "https://laravel.build/laravel-api?with=mysql" | bash
Windows (PowerShell): 
Invoke-WebRequest -Uri "https://laravel.build/laravel-api?with=mysql" -OutFile laravel-api.ps1; ./laravel-api.ps1

3 Menyalakan Environment Sail

Masuk ke folder project, lalu nyalakan Docker container menggunakan binary Sail bawaan:

cd laravel-api
./vendor/bin/sail up

Tips Pro: Tambahkan flag -d (contoh: ./vendor/bin/sail up -d) agar container berjalan di background mode, membebaskan terminal Anda untuk eksekusi perintah lain.
