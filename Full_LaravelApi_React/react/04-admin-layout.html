Membangun Wrapper Admin Layout & Sidebar Responsif

Seni menyusun kerangka dashboard aplikasi menggunakan teknik grid/flexbox, state toggle mobile sidebar, dan integrasi komponen Outlet untuk dynamic page loading.

LOGIKA KERJA NATIVE OUTLET ROUTER

Komponen Layout bertindak sebagai Shell (Cangkang) permanen. Ketika user berpindah link dari `/` ke `/guru`, area Sidebar dan Navbar tidak akan melakukan re-render ulang, melainkan hanya area <Outlet /> saja yang bertukar isi secara dinamis.

Sidebar (Tetap) + Navbar (Tetap) + [ Halaman Dinamis via Outlet ]

01 Master Wrapper Layout (`src/components/Layout.tsx`)

Gunakan layout flexbox dengan kombinasi state boolean untuk menangani aksi menyembunyikan dan memunculkan panel laci sidebar pada perangkat smartphone (mobile device). 

import React, { useState } from 'react';
import { Outlet } from 'react-router-dom';
import Sidebar from './Sidebar';
import Navbar from './Navbar';

export default function Layout(): React.JSX.Element {
  const [sidebarOpen, setSidebarOpen] = useState<boolean>(false);

  return (
    <div class="flex h-screen w-screen overflow-hidden bg-slate-50 text-slate-800">
      {/* 1. Komponen Sidebar Navigasi Kiri */}
      <Sidebar isOpen={sidebarOpen} setIsOpen={setSidebarOpen} />

      {/* 2. Container Konten Utama Sebelah Kanan */}
      <div class="flex flex-col flex-1 overflow-hidden">
        
        {/* Topbar/Navbar Kepala Halaman */}
        <Navbar setSidebarOpen={setSidebarOpen} />

        {/* Main Body: Tempat Sub-Materi Dipasang via Outlet */}
        <main class="flex-1 overflow-y-auto p-4 md:p-6 lg:p-8">
          <div class="max-w-6xl mx-auto">
            <Outlet /> {/* Konten dinamis (Dashboard / GuruList) dirender disini */}
          </div>
        </main>

      </div>
    </div>
  );
}

02 Komponen Menu Kiri (`src/components/Sidebar.tsx`)

Implementasikan utility Tailwind classes -translate-x-full md:translate-x-0 agar sidebar otomatis tersembunyi di layar kecil namun langsung tampil permanen pada monitor desktop. 

import React from 'react';
import { NavLink } from 'react-router-dom';
import { LayoutDashboard, Users, GraduationCap, X } from 'lucide-react'; // Menggunakan lucide icons

interface SidebarProps {
  isOpen: boolean;
  setIsOpen: (value: boolean) => void;
}

export default function Sidebar({ isOpen, setIsOpen }: SidebarProps): React.JSX.Element {
  
  const menuItems = [
    { name: 'Dashboard', path: '/', icon: LayoutDashboard },
    { name: 'Data Guru', path: '/guru', icon: GraduationCap },
    { name: 'Data Murid', path: '/murid', icon: Users },
  ];

  return (
    <>
      {/* Backdrop Hitam Transparan Khusus Layar Mobile Saat Sidebar Aktif */}
      {isOpen && (
        <div 
          onClick={() => setIsOpen(false)} 
          class="fixed inset-0 z-40 bg-slate-900/40 backdrop-blur-sm md:hidden"
        />
      )}

      {/* Panel Utama Sidebar */}
      <aside class={`fixed md:sticky top-0 bottom-0 left-0 z-50 flex flex-col w-64 h-full bg-slate-900 text-slate-300 transition-transform duration-300 transform 
        ${isOpen ? 'translate-x-0' : '-translate-x-full'} md:translate-x-0 border-r border-slate-800`}>
        
        {/* Header Identitas Sekolah / Aplikasi */}
        <div class="flex items-center justify-between h-16 px-6 border-b border-slate-800">
          <h1 class="text-lg font-bold text-white tracking-tight flex items-center gap-2">
            <span class="text-blue-500 font-extrabold">BUKU</span>GURU
          </h1>
          <button onClick={() => setIsOpen(false)} class="p-1 rounded text-slate-400 hover:text-white md:hidden">
            <X size={18} />
          </button>
        </div>

        {/* Looping Navigasi Menu */}
        <nav class="flex-1 px-4 py-6 space-y-1.5 overflow-y-auto">
          {menuItems.map((item) => {
            const Icon = item.icon;
            return (
              <NavLink
                key={item.path}
                to={item.path}
                onClick={() => setIsOpen(false)} // Otomatis tutup laci jika link diklik di HP
                class={({ isActive }) => `flex items-center gap-3 px-4 py-3 rounded-xl text-sm font-medium transition-all duration-200
                  ${isActive 
                    ? 'bg-blue-600 text-white shadow-md shadow-blue-600/10' 
                    : 'hover:bg-slate-800/60 hover:text-white'}`
                }
              >
                <Icon size={18} />
                <span>{item.name}</span>
              </NavLink>
            );
          })}
        </nav>
      </aside>
    </>
  );
}
