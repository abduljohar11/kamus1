Routing & State Management: React Router DOM & TanStack Query v5

Panduan instalasi dan arsitektur routing multi-page (SPA) serta optimalisasi data fetching, caching, dan sinkronisasi otomatis status server menggunakan React Query.

MENYELARASKAN NAVIGASI DAN SERVER STATE

Dalam aplikasi berskala produksi, penanganan navigasi halaman dan manajemen pemuatan data dari API harus dipisah secara efisien:

  • React Router DOM: Bertanggung jawab memetakan path URL browser ke komponen halaman yang sesuai tanpa memicu reload halaman penuh.
  • TanStack Query (React Query): Bertugas menghapus kebutuhan useEffect manual untuk fetching. Ia mengelola otomatis status loading, error, caching data, dan background refetching saat koneksi sempat terputus.

01 Instalasi & Setup React Router DOM v6

Jalankan perintah berikut di terminal proyek Anda untuk memasang router beserta tipe data pendukungnya:

npm install react-router-dom
Implementasi File Routing (`src/router/AppRouter.tsx`): 
import React from 'react';
import { BrowserRouter, Routes, Route, Navigate } from 'react-router-dom';
import Layout from '../components/Layout';
import Dashboard from '../pages/Dashboard';
import GuruList from '../pages/guru/GuruList';

export default function AppRouter(): React.JSX.Element {
  return (
    <BrowserRouter>
      <Routes>
        {/* Rute Publik */}
        <Route path="/login" element={<div>Halaman Login</div>} />

        {/* Rute Terproteksi Terbungkus Layout Sidebar */}
        <Route path="/" element={<Layout />}>
          <Route index element={<Dashboard />} />
          <Route path="guru" element={<GuruList />} />
        </Route>

        {/* Fallback jika route tidak ditemukan */}
        <Route path="*" element={<Navigate to="/" replace />} />
      </Routes>
    </BrowserRouter>
  );
}

02 Instalasi & Setup TanStack Query v5

Pasang core library React Query beserta komponen perangkat pencari kesalahan devtools untuk memantau status cache data secara real-time:

npm install @tanstack/react-query @tanstack/react-query-devtools
Inisialisasi Wrapper Global di Berkas Utama (`src/main.tsx`):

Bungkus rute aplikasi Anda menggunakan QueryClientProvider agar seluruh sub-halaman dapat menikmati fasilitas caching data:

import { StrictMode } from 'react';
import { createRoot } from 'react-dom/client';
import { QueryClient, QueryClientProvider } from '@tanstack/react-query';
import { ReactQueryDevtools } from '@tanstack/react-query-devtools';
import AppRouter from './router/AppRouter';
import './index.css';

// Membuat instance query client dengan konfigurasi default global
const queryClient = new QueryClient({
  defaultOptions: {
    queries: {
      refetchOnWindowFocus: false, // Menghindari re-fetch otomatis saat pindah tab browser
      retry: 1, // Batasi percobaan ulang request jika server error sebanyak 1 kali saja
      staleTime: 5 * 60 * 1000, // Data dianggap segar selama 5 menit sebelum boleh ditarik ulang
    },
  },
});

createRoot(document.getElementById('root')!).render(
  <StrictMode>
    <QueryClientProvider client={queryClient}>
      <AppRouter />
      {/* Devtools mempermudah inspeksi cache data API, hanya muncul pada mode development */}
      <ReactQueryDevtools initialIsOpen={false} />
    </QueryClientProvider>
  </StrictMode>
);

03 Integrasi TanStack Query dalam Logic Layer

Mari kita ubah custom hook lama kita (useGuruLogic.ts) agar memanfaatkan fitur useQuery. Kita tidak lagi membutuhkan useEffect atau state penampung manual.

File Baru yang Lebih Ringkas (`src/pages/guru/useGuruLogic.ts`): 
import { useQuery } from '@tanstack/react-query';
import { userApi } from '../../api/userApi';
import { User, ApiResponse } from '../../types/user';

export function useGuruLogic() {
  // useQuery otomatis menghandle state isLoading, data, dan error secara aman
  const { data, isLoading, isError, error, refetch } = useQuery<ApiResponse<User[]>, Error>({
    queryKey: ['users', 'guru'], // Key unik sebagai alamat pengenal laci penyimpanan cache
    queryFn: () => userApi.getAllUsers('guru'), // Fungsi eksekusi pemanggilan axios
  });

  return {
    guruList: data?.data || [], // Mengekstrak data array di dalam struktur ApiResponder
    isLoading,
    errorMessages: isError ? error?.message || 'Terjadi kekacauan sistem' : null,
    refreshData: refetch
  };
}

GOLDEN RULES NAVIGASI SPA

Saat membuat menu navigasi di komponen Sidebar atau Navbar, jangan pernah menggunakan tag HTML native <a href="..."> karena itu akan memaksa browser memuat ulang seluruh aset javascript dari nol. Gunakan komponen bawaan Link atau NavLink: 

import { NavLink } from 'react-router-dom';

export function Sidebar() {
  return (
    <nav class="flex flex-col gap-2">
      <NavLink 
        to="/guru" 
        className={({ isActive }) => `p-2 rounded-lg ${isActive ? 'bg-blue-600 text-white' : 'text-slate-400'}`}
      >
        Manajemen Data Guru
      </NavLink>
    </nav>
  );
}
