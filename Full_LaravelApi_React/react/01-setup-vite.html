Inisialisasi Project Frontend

Setup Environment: Vite, Tailwind CSS v4 & shadcn/ui

Panduan langkah demi langkah membangun pondasi aplikasi React Modern yang cepat, optimal, dan siap diintegrasikan dengan REST API backend.

MODERN FRONTEND STACK

Kita meninggalkan CRA (Create React App) yang lambat dan beralih ke ekosistem modern:

  • Vite: Build tool super cepat berbasis Esbuild untuk core aplikasi React.
  • Tailwind CSS v4: Engine baru yang lebih cepat dengan konfigurasi berbasis CSS native.
  • shadcn/ui: Komponen UI premium yang kodenya di-copy langsung ke dalam project kita (bukan node_modules dependency).

01 Buat Project React via Vite

Buka terminal pada environment Linux Mint atau OS Anda, lalu jalankan perintah instalasi Vite berikut:

Terminal CLI
# Buat project baru (pilih React & TypeScript/JavaScript sesuai preferensi)
npm create vite@latest ajocell-frontend -- --template react
# Masuk ke direktori dan install base dependency
cd ajocell-frontend
npm install

02 Install & Konfigurasi Tailwind CSS v4

Tailwind v4 hadir dengan arsitektur baru tanpa memerlukan file tailwind.config.js tersendiri. Referensi instalasi resmi dapat Anda pelajari langsung di Dokumentasi Resmi Tailwind Vite .

Langkah A: Jalankan Package v4 + Plugin Vite
npm install tailwindcss @tailwindcss/vite
Langkah B: Daftarkan Plugin di vite.config.js 
import { defineConfig } from 'vite'
import react from '@vitejs/react-swc'
import tailwindcss from '@tailwindcss/vite'

// https://vite.dev/config/
export default defineConfig({
  plugins: [
    react(),
    tailwindcss(), // Tambahkan plugin tailwind disini
  ],
})
Langkah C: Inject Direktif Baru di src/index.css

Hapus seluruh isi src/index.css bawaan dan ganti dengan satu baris import baru versi 4:

@import "tailwindcss";

03 Inisialisasi Komponen shadcn/ui

shadcn/ui membutuhkan konfigurasi path alias agar file komponen bisa di-generate secara otomatis ke dalam folder aplikasi kita.

Langkah A: Pasang Node Types (Untuk Keamanan Path)
npm install -D @types/node
Langkah B: Sesuaikan tsconfig.json / jsconfig.json

Tambahkan compiler options untuk alias path `@/` agar mengarah ke folder src:

{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./src/*"]
    }
  }
}
Langkah C: Jalankan CLI shadcn/ui Init
npx shadcn@latest init

* Ikuti instruksi wizard di terminal. Pilih tema warna default (Slate/Zinc) dan pastikan path alias diatur ke @/*.

CARA PENGGUNAAN KOMPONEN SHADCN

Ingat, shadcn/ui bukan library komponen biasa. Komponen dipasang per kebutuhan. Misalnya, jika Anda membutuhkan elemen Button premium, jalankan perintah di bawah ini:

npx shadcn@latest add button

File baru akan otomatis terbuat di dalam src/components/ui/button.jsx. Anda dapat menggunakannya langsung di file komponen Anda seperti ini: 

import { Button } from "@/components/ui/button"

export default function Dashboard() {
  return <Button variant="default">Simpan Data</Button>
}
