Delete Data dari Database

⚠️ Analogi: Proses Delete adalah seperti menghapus permanen sebuah file di komputer. Namun, di aplikasi web, kita harus memberikan "konfirmasi" (seperti kotak dialog Recycle Bin) agar pengguna tidak tidak sengaja menghapus data yang penting.

Langkah 1 - Menambahkan Method Destroy

Buka file app/Http/Controllers/PostController.php. Kita akan menambahkan method destroy yang akan menghapus record berdasarkan ID yang dikirim.

/**
 * destroy
 *
 * @param  mixed $post
 * @return void
 */
public function destroy(Post $post)
{
    // Melakukan penghapusan data
    $post->delete();

    // Redirect kembali ke index dengan pesan flash
    return redirect()->route('posts.index')->with('message', 'Data Berhasil Dihapus!');
}

Langkah 2 - Implementasi di React (Index.jsx)

Sekarang kita perlu menambahkan tombol hapus dan logika pengirimannya di file resources/js/Pages/Posts/Index.jsx. Kita menggunakan router.delete dari Inertia.

import React from 'react';
import Layout from '../../Layouts/Default';
import { Link, router } from '@inertiajs/react';

export default function PostIndex({ posts, flash }) {

    // Fungsi untuk menghapus data
    const deletePost = (id) => {
        if(confirm('Apakah Anda yakin ingin menghapus data ini?')) {
            router.delete(`/posts/${id}`);
        }
    }

    return (
        <Layout>
            <div className="p-4">
                {/* Flash Message */}
                {flash.message && (
                    <div className="bg-green-500 text-white p-3 rounded mb-4">{flash.message}</div>
                )}

                <table className="w-full bg-white rounded shadow">
                    <tbody>
                        {posts.map((post) => (
                            <tr key={post.id}>
                                <td>{post.title}</td>
                                <td className="text-center">
                                    <button 
                                        onClick={() => deletePost(post.id)} 
                                        className="bg-red-500 text-white px-3 py-1 rounded">
                                        DELETE
                                    </button>
                                </td>
                            </tr>
                        ))}
                    </tbody>
                </table>
            </div>
        </Layout>
    );
}

Langkah 3 - Uji Coba Hapus Data

Jalankan npm run dev. Pada halaman daftar post, klik tombol DELETE. Browser akan memunculkan konfirmasi; jika kamu menekan "OK", Inertia akan mengirimkan request ke server dan halaman akan terupdate secara otomatis tanpa refresh (SPA).
