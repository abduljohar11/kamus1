Konfigurasi Inertia Client Side

Setelah selesai dengan konfigurasi Server-side, sekarang kita perlu menyiapkan React dan Vite agar bisa merender komponen yang dikirimkan oleh Laravel.

Langkah 1 - Install Dependencies

Gunakan perintah berikut untuk menginstal package Inertia, React, dan Vite Plugin:

npm install @inertiajs/react react react-dom @vitejs/plugin-react

Langkah 2 - Inisialisasi App

Rename file resources/js/app.js menjadi resources/js/app.jsx dan masukkan kode berikut:

import './bootstrap';
import { createInertiaApp } from '@inertiajs/react'
import { createRoot } from 'react-dom/client'

createInertiaApp({
  resolve: name => {
    const pages = import.meta.glob('./Pages/**/*.jsx', { eager: true })
    return pages[`./Pages/${name}.jsx`]
  },
  setup({ el, App, props }) {
    createRoot(el).render(<App {...props} />)
  },
})

Jangan lupa membuat folder resources/js/Pages untuk menyimpan semua komponen halaman React kamu nantinya.

Langkah 3 - Konfigurasi Vite

Buka file vite.config.js dan tambahkan dukungan React:

import { defineConfig } from 'vite';
import laravel from 'laravel-vite-plugin';
import react from '@vitejs/plugin-react';

export default defineConfig({
    plugins: [
        laravel({
            input: 'resources/js/app.jsx',
            refresh: true,
        }),
        react(),
    ],
});

Langkah 4 - Update Environment

Pastikan file .env kamu merujuk pada URL yang benar (biasanya port 8000 untuk server php artisan serve):

APP_URL=http://localhost:8000

Langkah 5 - Compiling Asset

Jalankan perintah berikut di terminal baru untuk mulai melakukan proses build asset secara real-time:

npm run dev

Kesimpulan

Persiapan lingkungan kerja (Environment) sudah lengkap! Laravel sudah terhubung dengan React melalui Inertia.js dan dioptimasi oleh Vite. Di materi selanjutnya, kita akan belajar cara **Menampilkan Data dari Database** ke komponen React.
