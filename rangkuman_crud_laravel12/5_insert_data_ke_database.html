Insert Data ke Database

ðŸ“¦ Analogi: Proses Store data seperti mengirim paket. Form adalah paketnya, Validation adalah petugas sortir yang memastikan alamat dan isi paket lengkap, dan Model adalah kurir yang memasukkan paket tersebut ke dalam gudang besar bernama Database.

Langkah 1 - Menambahkan Method di Controller

Buka app/Http/Controllers/PostController.php dan tambahkan method create untuk menampilkan form, serta store untuk memproses datanya.

public function create()
{
    // Menampilkan halaman Create.jsx
    return inertia('Posts/Create');
}

public function store(Request $request)
{
    // 1. Validasi Input
    $request->validate([
        'title'   => 'required',
        'content' => 'required',
    ]);

    // 2. Simpan ke Database
    Post::create([
        'title'   => $request->title,
        'content' => $request->content
    ]);

    // 3. Redirect dengan pesan sukses
    return redirect()->route('posts.index')->with('message', 'Data Berhasil Disimpan!');
}
FieldValidasiKeterangan
titlerequiredJudul post tidak boleh kosong.
contentrequiredKonten post wajib diisi.

Langkah 2 - Membuat View Form (Create)

Buat file baru di resources/js/Pages/Posts/Create.jsx. Kita akan menggunakan React Hooks untuk mengelola state form.

import React, { useState } from 'react';
import Layout from '../../Layouts/Default';
import { router } from '@inertiajs/react';

export default function CreatePost({ errors }) {
    // Definisi State
    const [title, setTitle] = useState('');
    const [content, setContent] = useState('');

    // Fungsi Submit Form
    const storePost = async (e) => {
        e.preventDefault();
        
        router.post('/posts', {
            title: title,
            content: content
        });
    }

    return (
        <Layout>
            <div className="bg-white p-6 rounded shadow-md">
                <form onSubmit={storePost}>
                    <div className="mb-4">
                        <label className="block font-bold">TITLE</label>
                        <input type="text" className="w-full border p-2 rounded" value={title} onChange={(e) => setTitle(e.target.value)} />
                        {errors.title && <div className="text-red-500 mt-2">{errors.title}</div>}
                    </div>
                    <div className="mb-4">
                        <label className="block font-bold">CONTENT</label>
                        <textarea className="w-full border p-2 rounded" rows="5" value={content} onChange={(e) => setContent(e.target.value)}></textarea>
                        {errors.content && <div className="text-red-500 mt-2">{errors.content}</div>}
                    </div>
                    <button type="submit" className="bg-blue-600 text-white px-4 py-2 rounded">SAVE</button>
                </form>
            </div>
        </Layout>
    );
}

Langkah 3 - Uji Coba

Jalankan perintah npm run dev dan buka /posts/create di browser Anda. Cobalah tekan tombol SAVE tanpa mengisi apapun untuk melihat pesan error validasi.
