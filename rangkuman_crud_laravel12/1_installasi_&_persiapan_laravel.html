Instalasi & Persiapan Laravel

Apa itu Inertia.js?

Inertia.js merupakan adapter yang menghubungkan antara backend dan frontend tanpa perlu menggunakan REST API. Ini memungkinkan kita membangun aplikasi SPA (Single Page Application) dengan tetap merasakan pengalaman pengembangan aplikasi monolith.

Untuk penjelasan lebih detail, silakan baca artikel resminya di: inertiajs.com.

Langkah 1 - Persiapan Composer

Pastikan Composer sudah terinstall di komputer Anda. Kamu bisa mengeceknya dengan menjalankan perintah berikut di terminal:

composer --version

Langkah 2 - Membuat Project Laravel

Gunakan perintah di bawah ini untuk membuat project Laravel baru dengan nama inertia-react:

composer create-project laravel/laravel inertia-react

Langkah 3 - Konfigurasi Database

Buka file .env dan sesuaikan konfigurasi database Anda. Contoh untuk MySQL:

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=db_inertia_react
DB_USERNAME=root
DB_PASSWORD=

Langkah 4 - Migration Tabel Posts

Jalankan perintah ini untuk membuat Model sekaligus file Migration:

php artisan make:model Post -m

Edit file migration di folder database/migrations/ dan sesuaikan bagian up():

public function up(): void
{
    Schema::create('posts', function (Blueprint $table) {
        $table->id();
        $table->string('title');
        $table->text('content');
        $table->timestamps();
    });
}

Langkah 5 - Konfigurasi Model

Buka file app/Models/Post.php. Tambahkan properti $fillable untuk mengizinkan input data ke kolom tertentu:

<?php

namespace App\Models;

use Illuminate\Database\Eloquent\Model;

class Post extends Model
{
    protected $fillable = [
        'title',
        'content'
    ];
}

Langkah 6 - Menjalankan Migrate

Sinkronkan tabel ke database dengan perintah berikut:

php artisan migrate

Kesimpulan

Kita telah berhasil melakukan instalasi Laravel dan menyiapkan database. Di artikel selanjutnya, kita akan masuk ke tahap integrasi **Inertia.js Server-side**.
