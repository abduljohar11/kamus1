01 Laravel Dasar & Blade

Jangan pakai React dulu. Fokus menguasai bagaimana data berpindah dari URL ke layar.

  • Routing: Cara membuat URL.
  • Controller: Tempat menulis logika.
  • Eloquent: Cara mengambil data database.
// web.php (Routing Laravel)
Route::get('/guru', [GuruController::class, 'index']);

// GuruController.php
public function index() {
    $data = Guru::all(); // Ini Eloquent
    return view('guru.index', compact('data')); // Ini Blade
}

    return (
        <ul>

    );
